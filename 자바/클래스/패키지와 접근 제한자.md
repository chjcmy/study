## 패키지
##### 패키지의 물리적인 형태는 파일 시스템의 폴더이다.

---

패키지 선언
~~~java
package 상위패키지.하위패키지;
~~~

## 접근 제한자
| 아리랑      | 아리랑                                                 |
| ----------- | ------------------------------------------------------ |
| * public    | 외부 클래스가 자유롭게 사용할 수 있도록 한다           |
| * protected | 같은 패키지 또는 자식 클래스에서 사용할 수 있도록 한다 |
| * private   | 개인적인 것이라 외부에서 사용될 수 없다                |
| * default            | 같은 패키지에 소속된 클래스에서만 사용할 수 있도록 한다                                                        |
