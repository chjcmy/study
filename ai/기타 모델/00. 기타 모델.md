# 00. 기타 주요 머신러닝 모델

이 섹션에서는 선형 모델이나 트리 기반 모델과 다른 독특한 접근 방식을 사용하는 주요 머신러닝 알고리즘들을 다룹니다.

## 1. 주요 모델

-   **[[01. K-최근접 이웃 (KNN)]]**: 거리 기반의 인스턴스 기반 학습 모델로, '가까운 데이터는 비슷할 것이다'라는 단순한 가정에서 출발합니다.

-   **서포트 벡터 머신 (SVM)**: 데이터를 가장 잘 구분하는 최적의 경계선(초평면)을 찾는 데 중점을 둔 강력한 분류 및 회귀 모델입니다.

-   **나이브 베이즈 (Naive Bayes)**: 베이즈 정리에 기반한 확률적 분류기로, 특성들이 서로 독립이라는 '순진한(naive)' 가정을 통해 빠르고 효율적인 예측을 수행합니다.
