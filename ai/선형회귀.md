# 회귀(Regression) 정리

## 핵심 정의

회귀는 **지도 학습 모델**로, 연속형 목표 변수와 설명 특징(독립 변수) 간의 관계를 모델링하는 기계 학습 기법입니다.subtitle-3.txt​

## 회귀의 유형

## 단순 회귀 (Simple Regression)

- **단일 독립 변수**가 종속 변수를 추정subtitle-3.txt​
    
- 선형 또는 비선형 가능
    
- 예시: 엔진 크기 → CO2 배출량 예측subtitle-3.txt​
    

## 다중 회귀 (Multiple Regression)

- **두 개 이상의 독립 변수** 사용subtitle-3.txt​
    
- 선형 또는 비선형 관계 모두 가능
    
- 예시: 엔진 크기 + 실린더 수 → CO2 배출량 예측subtitle-3.txt​

- 회귀는 **연속형 값을 예측**할 때 사용하는 기법subtitle-3.txt​
    
- 단순 회귀 = 독립 변수 1개
    
- 다중 회귀 = 독립 변수 2개 이상
    
- 알고리즘 선택은 종속 변수의 데이터 특성과 최적 적합도에 따라 결정subtitle-3.txt​

## 궁금한점

- 왜 디비로 도 예측이 가능 한데 왜 회귀 모델을 만들어서 비교를 할까 ???

데이터베이스(DB)를 사용하여 과거 데이터를 분석하고 예측할 수 있지만, 회귀 모델을 만드는 이유는 더 정교하고 체계적인 예측을 가능하게 하기 때문입니다.

1. **패턴 인식**: 회귀 모델은 데이터 간의 관계를 수학적으로 표현하여 패턴을 인식합니다. 이를 통해 단순히 과거 데이터를 조회하는 것보다 더 깊이 있는 통찰을 제공합니다.
    
2. **예측의 일반화**: 회귀 모델은 새로운 데이터에 대해서도 예측을 할 수 있도록 훈련됩니다. 즉, 과거 데이터에 기반하여 미래의 상황을 예측할 수 있는 능력을 갖추게 됩니다. 데이터베이스에서 단순히 과거 데이터를 조회하는 것은 특정 상황에만 적용될 수 있지만, 회귀 모델은 다양한 상황에 적용할 수 있는 일반화된 예측을 제공합니다.

결국, 회귀 모델은 데이터베이스의 정보를 활용하여 더 나은 예측을 할 수 있도록 도와주는 도구라고 할 수 있다
