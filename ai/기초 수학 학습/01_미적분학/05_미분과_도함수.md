# 05 미분과 도함수: 순간의 변화를 포착하는 마법

## 1. 미분, 왜 필요할까요? (The Big Picture)

"미분"이라는 단어를 들으면 어떤 느낌이 드시나요? 아마 많은 분들이 복잡한 수식과 그래프를 떠올리며 머리 아파하실지도 모릅니다. 하지만 미분의 핵심 아이디어는 생각보다 훨씬 간단하고 직관적입니다.

**미분은 바로 '어떤 함수의 특정 지점에서의 순간적인 변화율' 또는 '그래프의 한 점에서의 접선의 기울기'를 구하는 과정입니다.**

- **순간 변화율**: 시시각각 변하는 대상의 특정 순간의 변화를 정확히 알고 싶을 때 사용합니다. 예를 들어, 달리는 자동차의 특정 순간의 속도, 시간에 따라 변하는 주식 가격의 특정 시점에서의 변화율 등을 구할 수 있습니다.
- **접선의 기울기**: 함수의 그래프가 특정 지점에서 얼마나 가파른지를 나타냅니다. 기울기가 양수이면 함수가 그 지점에서 증가하고 있다는 의미이고, 음수이면 감소하고 있다는 의미입니다. 기울기가 0이면 잠시 멈춰있는 상태(극점)일 수 있습니다.

AI, 특히 딥러닝에서 미분은 모델을 학습시키는 데 핵심적인 역할을 합니다. 모델의 예측이 얼마나 틀렸는지를 나타내는 '[[16_정보_이론_기초#^교차 엔트로피|손실 함수(Loss Function)]]'를 최소화하는 방향으로 모델의 파라미터(가중치)를 업데이트해야 하는데, 이때 각 파라미터에 대해 손실 함수를 미분하여 얻은 '[[08_편미분과_그라디언트|기울기(Gradient)]]'를 사용하기 때문입니다. ([[08_편미분과_그라디언트#^경사 하강법|경사 하강법]])

## 2. 평균 변화율에서 순간 변화율로

미분의 개념을 이해하기 위해 먼저 '평균 변화율'부터 시작해봅시다.

- **평균 변화율**: 어떤 함수 $f(x)$에서 $x$값이 $a$에서 $b$까지 변할 때, $y$값은 $f(a)$에서 $f(b)$까지 변합니다. 이때의 평균 변화율은 두 점 $(a, f(a))$와 $(b, f(b))$를 잇는 직선의 기울기와 같습니다.

  > 평균 변화율 = $\frac{y\text{값의 변화량}}{x\text{값의 변화량}} = \frac{f(b) - f(a)}{b - a}$

하지만 우리가 정말 알고 싶은 것은 특정 '순간'의 변화율입니다. 어떻게 구할 수 있을까요?

$b$를 $a$에 아주 아주 가깝게 보내면, 두 점 사이의 기울기는 $a$에서의 접선의 기울기에 가까워질 것입니다. 이처럼 '아주 아주 가깝게 보낸다'는 개념을 수학에서는 **'[[04_극한과_입실론-델타_논법|극한(Limit)]]'** 이라고 합니다.

## 3. 미분 계수와 도함수의 정의 (The Formal Stuff)

**미분 계수(Derivative at a point)** 는 함수 $f(x)$의 한 점 $x=a$에서의 순간 변화율을 의미하며, $f'(a)$ (f 프라임 a 라고 읽습니다) 로 표기합니다.

> $$f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$$

여기서 $h$는 $x$의 아주 작은 변화량을 의미합니다. $h$가 0에 가까워질 때의 평균 변화율의 극한값이 바로 $a$에서의 미분 계수, 즉 접선의 기울기입니다.

**도함수(Derivative Function)** 는 원래 함수 $f(x)$의 모든 점에서의 미분 계수를 함수로 나타낸 것입니다. 즉, $x$에 어떤 값을 넣으면 그 점에서의 접선의 기울기를 알려주는 새로운 함수 $f'(x)$를 구하는 과정, 또는 그 결과를 의미합니다.

> $$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

## 4. 기본적인 미분 공식 (Shortcuts!)

매번 극한을 계산하는 것은 번거롭습니다. 다행히도, 자주 사용되는 함수들에 대해서는 미분 공식이 존재합니다.

1.  **상수 함수 미분**: $f(x) = c$ ($c$는 상수)
    - $f'(x) = 0$ (상수는 변하지 않으므로 변화율이 0입니다.)

2.  **거듭제곱 공식 (Power Rule)**: $f(x) = x^n$ ($n$은 실수)
    - $f'(x) = nx^{n-1}$ (지수($n$)가 앞으로 내려와 곱해지고, 원래 지수에서 1을 뺍니다.)
    - 예시: $f(x) = x^3$ 이면 $f'(x) = 3x^2$
    - 예시: $f(x) = x$ (즉, $x^1$) 이면 $f'(x) = 1 \cdot x^0 = 1$

3.  **상수배 미분**: $f(x) = c \cdot g(x)$
    - $f'(x) = c \cdot g'(x)$ (함수에 곱해진 상수는 그대로 둡니다.)
    - 예시: $f(x) = 5x^3$ 이면 $f'(x) = 5(3x^2) = 15x^2$

4.  **합/차의 미분**: $f(x) = g(x) \pm h(x)$
    - $f'(x) = g'(x) \pm h'(x)$ (각각 미분한 후 더하거나 뺍니다.)
    - 예시: $f(x) = 2x^3 + 4x$ 이면 $f'(x) = 6x^2 + 4$

## 5. 곱의 미분법과 몫의 미분법

두 함수가 곱해져 있거나 나눠져 있을 때는 조금 더 복잡한 공식이 필요합니다.

1.  **곱의 미분법**: $f(x) = g(x)h(x)$
    - $f'(x) = g'(x)h(x) + g(x)h'(x)$ (앞에꺼 미분 * 뒤에꺼 그대로 + 앞에꺼 그대로 * 뒤에꺼 미분)

2.  **몫의 미분법**: $f(x) = g(x) / h(x)$
    - $$f'(x) = \frac{g'(x)h(x) - g(x)h'(x)}{[h(x)]^2}$$ (분모 제곱 분에, (분자 미분*분모 그대로) - (분자 그대로*분모 미분))
