# 03 확률 분포: 데이터의 패턴을 수학적으로 표현하기

## 1. 확률 변수와 확률 분포

-   **확률 변수 (Random Variable)**: 어떤 확률적 실험의 결과에 따라 값이 결정되는 변수입니다. (예: 주사위를 던져 나오는 눈)
-   **확률 분포 (Probability Distribution)**: 확률 변수가 가질 수 있는 모든 값과, 그 값이 나타날 확률을 함께 나타낸 함수 또는 표입니다. 즉, 데이터가 어떤 형태로 퍼져 있는지를 보여주는 **데이터의 패턴 지도**입니다.

## 2. 이산 확률 분포 (Discrete Probability Distributions)

확률 변수가 정수처럼 셀 수 있는 값을 가질 때의 분포입니다.

### 2.1. 베르누이 분포 (Bernoulli Distribution)

-   **정의**: 단 한 번의 시도에서 '성공' 또는 '실패' 두 가지 결과만 나오는 분포.
-   **예시**: 동전 한 개를 던지는 것, 이메일이 스팸인지 아닌지.
-   **AI 활용**: 클릭/비클릭, 구매/비구매 등 모든 이진 분류 문제의 기본 가정입니다.

### 2.2. 이항 분포 (Binomial Distribution)

-   **정의**: 성공 확률이 $p$인 베르누이 시행을 $n$번 독립적으로 반복했을 때, 성공이 $k$번 나타날 확률 분포.
    $$ P(X=k) = \binom{n}{k} p^k (1-p)^{n-k} $$
-   **예시**: 동전을 10번 던져서 앞면이 3번 나올 확률.
-   **AI 활용**: 클릭률, 구매 전환율 등 특정 행동을 할지/안 할지와 같은 이진 결과를 모델링하는 데 사용됩니다.

## 3. 연속 확률 분포 (Continuous Probability Distributions)

확률 변수가 키, 몸무게처럼 특정 범위 내의 어떤 실수 값도 가질 수 있을 때의 분포입니다.

### 3.1. 균등 분포 (Uniform Distribution)

-   **정의**: 특정 범위 내의 모든 값이 나타날 확률이 동일한 분포.
-   **예시**: 0과 1 사이에서 임의의 숫자를 뽑을 때.
-   **AI 활용**: 모델의 가중치를 초기화할 때, 특정 범위 내에서 균등하게 값을 샘플링하는 경우가 많습니다.

### 3.2. 정규 분포 (Normal Distribution) / 가우시안 분포 (Gaussian Distribution)

-   **정의**: 자연 현상과 사회 현상에서 가장 흔하게 나타나는 아름다운 '종 모양(bell curve)'의 연속 확률 분포. 평균($\mu$)을 중심으로 대칭이며, 표준편차($\sigma$)에 따라 퍼진 정도가 결정됩니다.
-   **중심 극한 정리 (Central Limit Theorem)**: "표본의 크기가 충분히 크다면, 원래 데이터의 분포와 상관없이 표본 평균의 분포는 정규 분포에 가까워진다"는 매우 중요한 정리입니다.
-   **AI 활용**:
    -   **데이터 분포 가정**: 많은 통계적 기법과 AI 모델은 데이터나 모델의 오차(noise)가 정규 분포를 따른다고 가정합니다.
    -   **가중치 초기화**: 신경망의 가중치를 정규 분포에서 샘플링하여 초기화하면 학습이 더 안정적으로 이루어지는 경향이 있습니다.
    -   **변분 오토인코더 (VAE)**: 데이터의 잠재 공간(latent space)이 정규 분포를 따르도록 학습하여 새로운 데이터를 생성하는 생성 모델의 핵심 아이디어입니다.

## 4. AI와의 연결점 요약

확률 분포는 AI에게 **데이터를 이해하는 언어**를 제공합니다.

-   **모델링**: 데이터가 어떤 분포를 따르는지 파악하여 그에 맞는 모델을 선택합니다. (예: 이진 결과 -> 베르누이/이항 분포, 일반적인 연속 값 -> 정규 분포)
-   **불확실성 표현**: 모델의 예측 결과를 단일 값이 아닌 확률 분포로 표현하여 예측의 불확실성을 정량화할 수 있습니다.
-   **생성 모델**: 특정 확률 분포를 학습하여, 그 분포로부터 새로운 데이터를 샘플링함으로써 진짜 같은 가짜 데이터(이미지, 텍스트 등)를 만들어냅니다.

## 5. 확인 문제 (Practice Problems)

1.  **분포 선택**: 다음 상황에 가장 적합한 확률 분포는 무엇일까요? (베르누이, 이항, 정규)
    -   (a) 한 명의 고객이 광고를 클릭할지 안 할지.
    -   (b) 100명의 고객 중 10명이 광고를 클릭할 확률.
    -   (c) 대한민국 성인 남성의 키 분포.

2.  **정규 분포**: 평균이 100이고 표준편차가 15인 정규 분포에서, 데이터는 어떤 값 근처에 가장 많이 모여 있을까요?

3.  **중심 극한 정리**: 주사위를 던져 나오는 눈의 분포는 1부터 6까지 동일한 확률을 가지는 균등 분포입니다. 만약 주사위를 100번 던져서 나온 눈의 평균을 구하는 실험을 수천 번 반복한다면, 그 평균들의 분포는 어떤 모양을 띨까요?

---
*확률 분포는 세상의 불확실하고 무작위적으로 보이는 데이터 속에서 패턴과 질서를 찾아내는 수학적 도구이며, AI가 데이터를 이해하고 미래를 예측하는 능력의 근간을 이룹니다.*
