# 03 적분 (Integration): 불확실성을 정량화하는 언어

## 1. 적분의 핵심 아이디어: 넓이와 총합

적분은 미분과 반대로, 아주 작은 조각들을 모두 더하여 전체의 양을 구하는 과정입니다. 기하학적으로는 **'함수 그래프 아래의 넓이'** 를 구하는 것과 같습니다.

-   **정적분 (Definite Integral)**: $\int_{a}^{b} f(x) dx$
    -   **의미**: 함수 $f(x)$의 그래프 아래, $x=a$부터 $x=b$까지의 넓이를 구합니다.
    -   **결과**: 특정 구간의 넓이이므로, 하나의 **상수 값**으로 나옵니다.

## 2. AI와의 연결점: 확률, 기댓값, 그리고 모델 평가

적분은 모델 학습 과정에서 미분만큼 직접적으로 사용되지는 않지만, AI의 또 다른 기둥인 **확률 이론**의 근간을 이루며, 모델의 성능을 평가하는 데에도 사용됩니다.

### 1. 확률 계산 (Calculating Probability)

-   **확률 밀도 함수 (Probability Density Function, PDF)**: 키, 몸무게, 시간처럼 연속적인 값을 갖는 데이터의 분포를 나타내는 함수입니다. (예: 정규 분포 곡선)
-   **확률은 곧 넓이**: 연속적인 데이터에서 "키가 정확히 175.0000...cm일 확률"은 0입니다. 대신, **"키가 170cm에서 180cm 사이일 확률"** 처럼 특정 **구간**에 속할 확률을 계산합니다. 이 확률은 해당 구간에서 **확률 밀도 함수 그래프 아래의 넓이**와 정확히 같습니다.
-   **넓이 계산**: 이 넓이를 구하는 유일한 방법이 바로 **적분**입니다.
    $$ P(170 \le X \le 180) = \int_{170}^{180} f(x) dx $$
    여기서 $f(x)$는 키 데이터의 확률 밀도 함수입니다.

### 2. 기댓값 계산 (Calculating Expected Value)

-   **기댓값(Expected Value)**: 어떤 확률 분포에서 평균적으로 기대할 수 있는 값입니다. 즉, 확률까지 고려한 가중 평균입니다.
-   **연속 변수의 기댓값**: 연속 확률 변수 $X$의 기댓값 $E[X]$는 각 값 $x$에 그 값이 나타날 확률 밀도 $f(x)$를 곱한 것을 모든 가능한 $x$에 대해 적분하여 구합니다.
    $$ E[X] = \int_{-\infty}^{\infty} x f(x) dx $$
    기댓값은 강화 학습의 보상 계산, 생성 모델의 샘플링 등 AI의 여러 분야에서 중요한 개념입니다.

### 3. 모델 성능 평가 (Model Evaluation)

-   **AUC (Area Under the Curve)**: 분류 모델의 성능을 평가하는 중요한 지표 중 하나인 **ROC 곡선(ROC Curve)** 아래의 넓이를 의미합니다.
    -   ROC 곡선은 모델의 임계값(threshold) 변화에 따라 분류 결과가 어떻게 변하는지를 보여주는 그래프입니다.
    -   AUC 값은 0과 1 사이이며, 1에 가까울수록 모델의 성능이 좋음을 의미합니다. 이 넓이를 계산하는 데 적분이 사용됩니다.

따라서, **적분은 AI가 데이터를 모델링하고 불확실성을 수학적으로 다루며, 모델의 성능을 종합적으로 평가하는 데 필수적인 확률 이론의 언어**입니다.

## 3. 확인 문제 (Practice Problems)

1.  **확률과 적분**: 어떤 연속 확률 변수의 확률 밀도 함수(PDF)가 주어졌을 때, 이 변수가 특정 범위 $[a, b]$ 안에 있을 확률을 구하려면 무엇을 계산해야 하나요?

2.  **기댓값**: 주사위를 던질 때 나오는 눈의 기댓값은 3.5입니다. 만약 0에서 6 사이의 값을 가지는 어떤 연속 확률 변수의 기댓값을 구하려면 어떤 연산이 필요한가요?

3.  **AUC**: 어떤 분류 모델의 AUC 점수가 0.95가 나왔습니다. 이 점수는 무엇을 의미하며, 이 값을 계산하는 데 사용되는 수학적 개념은 무엇인가요?
