# 00. 비지도 학습 (Unsupervised Learning)

이 섹션에서는 레이블이 없는 데이터에서 숨겨진 패턴이나 구조를 찾아내는 **비지도 학습(Unsupervised Learning)** 알고리즘들을 다룹니다.

## 1. 주요 비지도 학습 알고리즘

-   **[[01. 클러스터링 알고리즘]]**: 유사성을 기반으로 데이터 포인트를 자동으로 그룹화하여 클러스터를 형성합니다.

-   **차원 축소 (Dimensionality Reduction)**: 데이터의 특징(차원) 수를 줄여 데이터를 더 효율적으로 표현하고 시각화하며, 모델의 성능을 향상시키는 기법입니다. (예: PCA)

-   **연관 규칙 학습 (Association Rule Learning)**: 대규모 데이터셋에서 항목들 간의 흥미로운 관계를 찾아냅니다. (예: 장바구니 분석)
