# 차원 축소 알고리즘 정리

## 차원 축소 (Dimensionality Reduction) 개요

### 정의
중요한 데이터 세트 정보를 희생하지 않으면서 데이터 세트 기능의 수를 줄이는 과정입니다.

### 필요성
-   고차원 데이터는 분석 및 시각화하기 어려움
-   머신러닝 모델의 데이터세트를 단순화

### 작동 방식
원래 차원을 변환하여 새로운 특징을 생성합니다.

## 주요 차원 축소 알고리즘

### 1. 주성분 분석 (PCA: Principal Component Analysis)

#### 정의
데이터셋 특징이 선형적으로 상관되어 있다고 가정하는 **선형 차원 축소 알고리즘**입니다.

#### 목적
정보 손실을 최소화하면서 데이터를 단순화하고, 차원을 줄이고, 노이즈를 줄입니다.

#### 작동 방식
-   데이터셋 특징을 주성분(Principal Components)이라고 하는 상관관계가 없는 새로운 변수 집합으로 변환합니다.
-   주성분은 서로 직교하며 특징 공간의 새 좌표계를 정의합니다.
-   주성분은 중요도(설명하는 분산의 정도)가 높은 순서로 구성됩니다.
-   처음 몇 개의 구성요소에는 대부분의 정보가 포함되며, 나머지는 잡음을 나타내는 경향이 있습니다.

### 2. T-분산 확률적 이웃 임베딩 (t-SNE: T-distributed Stochastic Neighbor Embedding)

#### 정의
고차원 데이터 포인트를 저차원 공간(2차원 또는 3차원)에 매핑하는 비선형 차원 축소 알고리즘입니다.

#### 활용
-   복잡한 고차원 데이터에서 클러스터를 찾는 데 유용합니다.
-   이미지 및 텍스트와 같은 데이터에서도 잘 작동합니다.

#### 작동 방식
-   서로 가까운 점의 유사성을 보존하는 데 초점을 맞추고, 먼 지점에서는 그렇지 않습니다.
-   유사성은 점 쌍 사이의 거리를 사용하여 근접도로 측정됩니다.

#### 단점
-   확장성이 떨어집니다.
-   하이퍼파라미터에 민감하여 조정하기 어려울 수 있습니다.

### 3. 균일 매니폴드 근사 및 투영 (UMAP: Uniform Manifold Approximation and Projection)

#### 정의
t-SNE의 대안으로 자주 사용되는 **비선형 차원 축소 알고리즘**입니다.

#### 기반 이론
데이터가 고차원 공간에 내장된 저차원 매니폴드에 있다고 가정하는 매니폴드 이론을 기반으로 합니다.

#### 작동 방식
-   데이터의 고차원 그래프 표현을 구성합니다.
-   원본 데이터의 점 간 관계를 가장 잘 보존하는 저차원 그래프 구조를 최적화합니다.

#### 장점 (t-SNE 대비)
-   t-SNE보다 확장성이 뛰어납니다.
-   데이터의 하위 구조 외에도 글로벌 구조를 보존하여 t-SNE보다 클러스터링 성능이 더 높은 경우가 많습니다.

## 알고리즘 비교 예시 (Scikit-Learn MakeBlobs 시뮬레이션 데이터)

### 데이터셋
-   Scikit-Learn의 MakeBlobs 함수를 사용한 3D 시뮬레이션 데이터.
-   노란색, 보라색 클러스터 사이에 약간의 겹침이 있고, 다른 두 클러스터는 뚜렷하게 구분됩니다.

### 결과 비교
-   **PCA**: 블롭을 효과적으로 분리했습니다. 블롭이 선형적으로 상관되어 있으므로 PCA가 잘 작동할 것으로 예상되었습니다.
-   **t-SNE**: 데이터를 네 개의 클러스터로 클러스터링했으며 대부분 잘 분리했습니다. 보라색 군집 내에 녹색과 노란색 군집 중에서 레이블이 잘못 지정된 여러 점이 있었습니다.
-   **UMAP**: 클러스터 중 3개가 완전히 분리되지 않았습니다 (특히 노란색과 녹색 클러스터가 보라색 클러스터와 약간 겹침). 3D 입력 데이터에서 노란색과 보라색 클러스터가 완전히 분리되지 않았기 때문입니다. 이 예시에서는 t-SNE가 4개의 매우 뚜렷한 클러스터를 식별했기 때문에 UMAP보다 약간 더 나은 성능을 보였습니다.

## 핵심 요약
-   차원 축소 알고리즘은 중요한 정보 손실 없이 데이터셋 기능 수를 줄입니다.
-   주요 유형: [[PCA]], [[t-SNE]], [[UMAP]]
-   [[PCA]]는 선형 차원 축소로 정보 손실을 최소화하며 데이터를 단순화합니다.
-   [[t-SNE]]는 고차원 데이터를 저차원 공간에 매핑하여 클러스터를 찾습니다.
-   [[UMAP]]은 매니폴드 이론 기반의 비선형 차원 축소로, t-SNE보다 확장성이 뛰어나고 글로벌 구조를 잘 보존합니다.
