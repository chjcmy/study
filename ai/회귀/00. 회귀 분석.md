# 회귀 분석 (Regression Analysis)

## 1. 핵심 정의

**회귀 분석(Regression Analysis)**은 **지도 학습(Supervised Learning)**의 한 종류로, 하나 이상의 **독립 변수(설명 변수)**와 **종속 변수(목표 변수)** 간의 관계를 모델링하여 **연속적인 값을 예측**하는 기계 학습 기법입니다.

쉽게 말해, 어떤 원인(독립 변수)이 특정 결과(종속 변수)에 얼마나 영향을 미치는지 파악하고, 이를 통해 미래의 결과를 예측하는 방법론입니다.

## 2. 회귀의 종류

회귀 분석은 모델링하는 관계의 형태와 독립 변수의 개수에 따라 다양하게 나뉩니다.

-   **[[01. 단순 선형 회귀]]**: **단 하나의 독립 변수**를 사용하여 종속 변수와의 **직선 관계**를 모델링합니다.
    -   예시: `엔진 크기`만으로 자동차의 `CO2 배출량`을 예측합니다.
-   **[[02. 다중 선형 회귀]]**: **두 개 이상의 독립 변수**를 사용하여 종속 변수와의 **선형 관계**를 모델링합니다.
    -   예시: `엔진 크기`, `실린더 수`, `차량 무게` 등 여러 요인을 종합하여 `CO2 배출량`을 예측합니다.
-   **[[03. 다항 회귀]]**: 독립 변수를 거듭제곱하여 **곡선 관계**를 모델링하지만, 본질적으로는 선형 회귀의 확장입니다.
    -   예시: 시간에 따라 포물선 형태로 변하는 물체의 위치를 예측합니다.
-   **[[04. 비선형 회귀]]**: 모델의 파라미터 자체가 비선형적으로 결합된, 더 복잡한 관계를 모델링합니다.
    -   예시: 인구 증가나 약물 반응과 같은 S자 형태의 성장 곡선을 예측합니다.

## 3. 주요 응용 분야

회귀 분석은 다양한 분야에서 미래 예측 및 요인 분석을 위해 활용됩니다.

-   **비즈니스**:
    -   **매출 예측**: 광고비, 프로모션 기간, 시장 점유율 등을 기반으로 미래의 매출을 예측합니다.
    -   **부동산 가격 예측**: 주택의 크기, 위치, 침실 수, 건축 연도 등을 기반으로 가격을 예측합니다.
-   **환경 및 공중보건**:
    -   **강우량 예측**: 기온, 습도, 풍속, 기압 등을 기반으로 강우량을 예측합니다.
    -   **질병 발생 가능성 예측**: 환자의 나이, BMI, 혈압, 콜레스테롤 수치 등을 기반으로 특정 질병의 발생 가능성을 예측합니다.

## 4. 회귀 모델을 사용하는 이유

데이터베이스(DB)만으로도 과거 데이터를 조회하고 분석할 수 있지만, 회귀 모델을 사용하는 이유는 더 정교하고 일반화된 예측을 하기 위함입니다.

1.  **패턴 인식**: 회귀 모델은 데이터 간의 복잡한 관계를 수학적 함수로 표현하여, 사람이 직관적으로 파악하기 어려운 패턴을 인식합니다.
2.  **예측의 일반화**: 회귀 모델은 학습 데이터(과거)를 통해 학습한 패턴을 기반으로, 한 번도 본 적 없는 새로운 데이터(미래)에 대해서도 예측을 수행할 수 있는 **일반화 능력**을 갖추게 됩니다.
3.  **What-If 시나리오 분석**: "만약 광고비를 10% 늘린다면, 예상 매출은 얼마나 증가할까?"와 같이, 특정 변수를 조정했을 때 결과가 어떻게 변할지 시뮬레이션해볼 수 있습니다.
