# 4부: 컴파일과 최적화

> 예상 학습 시간: 10~12시간 (2개 장)
> "JVM 밑바닥까지 파헤치기" (深入理解Java虚拟机 3판, 저우즈밍)

자바 코드가 `.java` 파일에서 기계어까지 도달하는 여정에는 **두 번의 컴파일**이 존재한다. 프런트엔드 컴파일러(`javac`)가 소스를 바이트코드로 변환하고, 백엔드 컴파일러(JIT/AOT)가 바이트코드를 네이티브 코드로 변환한다.

- [11장: 백엔드 컴파일과 최적화](ch11/README.md)
