# 이넘 클래스 (Enum Class)
#Kotlin/Concept

> [!INFO] 이넘(Enum)이란?
> **이넘(Enum, 열거형) 클래스**는 서로 관련 있는 **상수(constant)들의 집합**을 정의하는 특별한 종류의 클래스입니다.
>
> 문자열이나 숫자 대신, 의미가 명확한 이름을 가진 상수를 사용함으로써 코드의 **가독성**과 **타입 안정성**을 크게 향상시킬 수 있습니다.

## 1. 기본 이넘 클래스

`enum class` 키워드를 사용하여 정의하며, 내부에 상수들을 쉼표로 구분하여 나열합니다.

```kotlin
enum class Direction {
    NORTH, SOUTH, WEST, EAST
}
```

각 이넘 상수는 `Direction` 타입의 객체입니다.

```kotlin
val dir: Direction = Direction.NORTH
println(dir) // 출력: NORTH
```

## 2. `when`과 함께 사용하기

이넘은 `when` 표현식과 함께 사용할 때 특히 강력합니다. `when`을 사용할 때 모든 이넘 상수를 처리하면 `else` 분기를 생략할 수 있어, 컴파일러가 모든 경우를 처리했는지 검사해 줄 수 있습니다.

```kotlin
fun getDirectionName(direction: Direction): String {
    return when (direction) {
        Direction.NORTH -> "북"
        Direction.SOUTH -> "남"
        Direction.WEST -> "서"
        Direction.EAST -> "동"
        // 모든 상수를 처리했으므로 else가 필요 없음
    }
}

println(getDirectionName(Direction.WEST)) // 출력: 서
```

## 3. 프로퍼티와 메소드를 갖는 이넘

이넘 클래스도 일반 클래스처럼 생성자와 프로퍼티, 메소드를 가질 수 있습니다.

```kotlin
enum class HttpStatus(val code: Int) { // 주 생성자에 프로퍼티 선언
    OK(200),
    NOT_FOUND(404),
    INTERNAL_SERVER_ERROR(500); // 프로퍼티나 메소드가 있으면 세미콜론(;) 필수

    // 메소드 정의
    fun isSuccess(): Boolean {
        return code in 200..299
    }
}

fun main() {
    val status = HttpStatus.OK
    println("Status: ${status.name}, Code: ${status.code}") // 출력: Status: OK, Code: 200
    println("Is success? ${status.isSuccess()}")           // 출력: Is success? true
}
```

## 4. 이넘의 기본 프로퍼티와 메소드

모든 이넘 클래스는 컴파일러가 자동으로 몇 가지 유용한 프로퍼티와 메소드를 추가해 줍니다.

-   **`name`**: 이넘 상수의 이름을 `String`으로 반환합니다. (예: `"NORTH"`)
-   **`ordinal`**: 이넘 상수가 선언된 순서를 `Int`로 반환합니다. (0부터 시작)
-   **`values()`**: 모든 이넘 상수를 포함하는 배열을 반환합니다.
-   **`valueOf(name: String)`**: 주어진 이름과 일치하는 이넘 상수를 반환합니다.

```kotlin
// 모든 방향을 순회하며 출력
for (direction in Direction.values()) {
    println("${direction.name} (ordinal: ${direction.ordinal})")
}

// 문자열로부터 이넘 상수 찾기
val west = Direction.valueOf("WEST")
println(west) // 출력: WEST
```
