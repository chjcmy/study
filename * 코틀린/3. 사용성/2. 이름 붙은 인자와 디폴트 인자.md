# 이름 붙은 인자와 디폴트 인자
#Kotlin/Function

> [!INFO]
> 코틀린은 **이름 붙은 인자(Named Arguments)**와 **디폴트 인자(Default Arguments)** 기능을 제공하여, 자바에 비해 훨씬 더 유연하고 가독성 높은 함수 호출을 가능하게 합니다.

## 1. 이름 붙은 인자 (Named Arguments)

-   **정의**: 함수를 호출할 때, **`파라미터 이름 = 값`** 형태로 명시적으로 인자를 전달하는 방식입니다.
-   **장점**:
    -   **가독성 향상**: 각 값이 어떤 파라미터에 전달되는지 명확하게 알 수 있습니다. 특히 `Boolean`이나 `null` 값을 전달할 때 유용합니다.
    -   **순서 독립성**: 파라미터의 순서를 지키지 않고 자유롭게 인자를 전달할 수 있습니다.

```kotlin
fun createButton(text: String, width: Int, height: Int, isVisible: Boolean) {
    // ...
}

fun main() {
    // 이름 붙은 인자를 사용하지 않은 경우 -> 각 값이 무엇을 의미하는지 알기 어려움
    createButton("Submit", 200, 50, true)

    // 이름 붙은 인자를 사용한 경우 -> 가독성이 매우 향상됨
    createButton(
        text = "Submit",
        width = 200,
        height = 50,
        isVisible = true
    )

    // 순서를 바꿔도 무방함
    createButton(
        height = 50,
        isVisible = true,
        text = "Cancel",
        width = 150
    )
}
```

> [!NOTE]
> 일반 인자와 이름 붙은 인자를 섞어 쓸 경우, **일반 인자가 모두 앞에 위치**해야 합니다.
> `createButton("Submit", width = 200, height = 50, true)` // 오류!

## 2. 디폴트 인자 (Default Arguments)

-   **정의**: 함수를 선언할 때, 파라미터에 **기본값**을 지정하는 기능입니다.
-   **장점**:
    -   함수 호출 시 값이 전달되지 않으면, 설정된 기본값이 사용됩니다.
    -   인자의 개수만 다른 여러 개의 함수를 만들어야 하는 **오버로딩(Overloading)을 줄일 수 있습니다.**

```kotlin
fun sendEmail(
    to: String,
    subject: String,
    body: String,
    isHtml: Boolean = false, // 기본값을 false로 지정
    priority: Int = 3        // 기본값을 3으로 지정
) {
    // ...
}

fun main() {
    // to, subject, body만 전달. isHtml과 priority는 기본값 사용
    sendEmail(
        to = "test@example.com",
        subject = "Hello",
        body = "This is a test email."
    )

    // isHtml 값만 명시적으로 전달
    sendEmail(
        to = "test@example.com",
        subject = "Hello",
        body = "<h1>Title</h1>",
        isHtml = true
    )
}
```

## 3. 이름 붙은 인자와 디폴트 인자의 조합

이 두 기능을 함께 사용하면, 원하는 파라미터의 값만 순서에 상관없이 지정할 수 있어 함수의 유연성이 극대화됩니다.

```kotlin
fun createUser(
    name: String,
    age: Int = 18,
    city: String = "Unknown",
    isVerified: Boolean = false
) {
    // ...
}

fun main() {
    // name만 필수적으로 전달하고, 나머지는 기본값 사용
    createUser("Alice")

    // name과 city만 지정하고 싶을 때, 이름 붙은 인자를 사용하여 age를 건너뜀
    createUser(name = "Bob", city = "New York")
}
```
