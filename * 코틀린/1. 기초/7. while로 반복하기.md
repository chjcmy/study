# while 반복문
#Kotlin/Basic #Kotlin/ControlFlow

> [!INFO] `while` 반복문이란?
> `while` 루프는 특정 조건이 `true`인 동안 코드 블록을 계속해서 반복 실행하는 제어문입니다. 주로 반복 횟수가 정해져 있지 않고, 특정 조건을 만족할 때까지 계속 작업을 수행해야 할 때 사용됩니다.

## 1. `while` 루프

-   루프에 진입하기 **전**에 조건을 먼저 검사합니다.
-   조건이 처음부터 `false`이면, 루프 본문은 한 번도 실행되지 않을 수 있습니다.

### 기본 구조
```kotlin
while (조건식) {
    // 조건식이 true인 동안 반복할 코드
}
```

### 예제
```kotlin
var i = 0
while (i < 5) {
    print("$i ")
    i++ // 조건식을 false로 만들 변화가 내부에 반드시 있어야 함
}
// 출력: 0 1 2 3 4
```

## 2. `do-while` 루프

-   루프 본문을 **먼저 한 번 실행**한 뒤, 조건을 검사합니다.
-   조건이 `false`이더라도, 루프 본문은 **최소 한 번은 실행**되는 것을 보장합니다.

### 기본 구조
```kotlin
do {
    // 반복할 코드
} while (조건식)
```

### 예제
```kotlin
var input: String
do {
    print("텍스트를 입력하세요 (종료하려면 'exit'): ")
    input = readlnOrNull() ?: ""
    println("입력한 내용: $input")
} while (input != "exit")

println("프로그램을 종료합니다.")
```

## 3. 루프 제어를 위한 연산자

`while` 루프 내에서 카운터 변수를 제어하기 위해 다음과 같은 연산자들이 자주 사용됩니다.

-   **복합 대입 연산자**: `+=`, `-=`, `*=`, `/=`
    ```kotlin
    i += 2 // i = i + 2 와 동일
    ```
-   **증감 연산자**: `++` (1 증가), `--` (1 감소)
    -   **전위(prefix)**: `++i` -> 값을 먼저 증가시킨 후, 그 결과를 사용
    -   **후위(postfix)**: `i++` -> 현재 값을 먼저 사용한 후, 값을 증가

    ```kotlin
    var a = 1
    println(++a) // 출력: 2 (a가 2가 됨)

    var b = 1
    println(b++) // 출력: 1 (b는 2가 됨)
    println(b)   // 출력: 2
    ```

> [!TIP] for vs while
> - **`for` 루프**: 반복 횟수가 명확하거나, 특정 범위(range) 또는 컬렉션의 모든 원소를 순회할 때 더 적합하고 안전합니다.
> - **`while` 루프**: 반복 횟수를 예측할 수 없고, 특정 조건에 따라 반복을 계속해야 할 때 유용합니다.
