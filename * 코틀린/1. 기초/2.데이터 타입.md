# 코틀린 데이터 타입
#Kotlin/Basic #Kotlin/TypeSystem

> [!INFO] 코틀린의 타입 시스템
> 코틀린은 **정적 타입 언어(Statically-typed language)**입니다. 이는 모든 변수와 표현식의 타입이 **컴파일 시점**에 결정된다는 의미이며, 이를 통해 타입 관련 오류를 런타임이 아닌 개발 단계에서 미리 발견할 수 있어 코드의 안정성이 높아집니다.

## 1. 숫자 타입 (Number Types)

-   코틀린의 모든 숫자 타입은 객체입니다.
-   **정수 타입**: `Byte`(8bit), `Short`(16bit), `Int`(32bit), `Long`(64bit)
-   **부동소수점 타입**: `Float`(32bit), `Double`(64bit)

```kotlin
val intNum: Int = 42
val longNum: Long = 42L      // Long 타입은 숫자 뒤에 'L'을 붙입니다.
val floatNum: Float = 3.14F    // Float 타입은 숫자 뒤에 'F'를 붙입니다.
val doubleNum: Double = 3.14   // 기본 소수점 리터럴은 Double 입니다.
```

> [!IMPORTANT] 명시적 타입 변환
> 코틀린은 작은 타입에서 큰 타입으로의 **자동 형 변환을 지원하지 않습니다.** 반드시 명시적으로 변환 함수를 호출해야 합니다.
> ```kotlin
> val num: Int = 42
> // val converted: Long = num // 컴파일 오류!
> val converted: Long = num.toLong() // OK
> ```

## 2. 문자 타입 (Char)

-   하나의 문자를 나타내며, 작은따옴표(`'`)로 감쌉니다.
-   숫자로 직접 취급되지 않으므로, 숫자로 변환하려면 명시적 변환이 필요합니다.

```kotlin
val aChar: Char = 'A'
// val code: Int = aChar // 컴파일 오류!
val code: Int = aChar.code // 'A'의 아스키 코드 값인 65를 반환
```

## 3. 문자열 타입 (String)

-   문자의 나열을 나타내며, 큰따옴표(`"`)로 감쌉니다.
-   **문자열 템플릿(String Template)**을 지원하여 문자열 안에 변수나 표현식을 쉽게 포함시킬 수 있습니다.

```kotlin
val name = "Kotlin"
val message = "Hello, $name!" // '$' 기호를 사용하여 변수 삽입
println(message) // 출력: Hello, Kotlin!

val text = "Price is ${1000 * 0.9}" // '${...}'를 사용하여 표현식 삽입
println(text) // 출력: Price is 900.0
```

## 4. 불리언 타입 (Boolean)

-   `true` 또는 `false` 두 가지 값만 가집니다.
-   `||`(OR), `&&`(AND), `!`(NOT) 논리 연산자와 함께 사용됩니다.

```kotlin
val isKotlinFun: Boolean = true
val isHard: Boolean = false
println(isKotlinFun && isHard) // 출력: false
```

## 5. 특수 목적 타입

-   **`Any`**: 코틀린의 모든 non-nullable 타입의 최상위 슈퍼클래스입니다. 자바의 `Object`와 유사합니다.
-   **`Unit`**: 의미 있는 값을 반환하지 않는 함수의 반환 타입을 명시할 때 사용합니다. 자바의 `void`와 유사하지만, `Unit`은 싱글턴 객체라는 차이가 있습니다. 함수에서 반환 타입 생략 시 컴파일러가 자동으로 `Unit`을 추론합니다.
    ```kotlin
    fun sayHello(): Unit { // ': Unit'은 생략 가능
        println("Hello")
    }
    ```
-   **`Nothing`**: 함수가 정상적으로 값을 반환하는 일이 없음을 명시하는 타입입니다. 주로 예외를 던지거나 무한 루프에 빠지는 함수의 반환 타입으로 사용됩니다.
    ```kotlin
    fun fail(message: String): Nothing {
        throw IllegalArgumentException(message)
    }
    ```
