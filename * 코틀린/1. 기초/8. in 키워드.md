# for 루프와 in 키워드
#Kotlin/Basic #Kotlin/ControlFlow

> [!INFO] `for`와 `in`
> 코틀린에서 `for` 루프는 `in` 키워드와 함께 사용되어, 특정 범위(Range)나 컬렉션에 포함된 모든 원소를 순서대로 순회(iterate)하는 데 사용됩니다.

## 1. `for` 루프와 범위(Range)

`..` 연산자를 사용하여 숫자 범위를 만들고, `for` 루프를 통해 해당 범위의 모든 숫자를 순회할 수 있습니다.

```kotlin
// 1부터 3까지 순서대로 출력
for (i in 1..3) {
    println("Hello $i!")
}
```
> **출력:**
> Hello 1!
> Hello 2!
> Hello 3!

### 다양한 범위 생성 방법

| 구문 | 설명 | 예시 | 결과 |
| :--- | :--- | :--- | :--- |
| `a..b` | `a`부터 `b`까지 **포함** (이상, 이하) | `1..5` | 1, 2, 3, 4, 5 |
| `a until b` | `a`부터 `b` **직전**까지 (b는 미포함) | `1 until 5` | 1, 2, 3, 4 |
| `a downTo b` | `a`부터 `b`까지 **감소** | `5 downTo 1` | 5, 4, 3, 2, 1 |
| `step n` | 값의 **간격**을 `n`으로 지정 | `1..10 step 2` | 1, 3, 5, 7, 9 |

```kotlin
// 10부터 0까지 2씩 감소하며 출력
for (i in 10 downTo 0 step 2) {
    print("$i ") // 출력: 10 8 6 4 2 0
}
```

문자(Char) 범위에도 동일하게 적용할 수 있습니다.
```kotlin
for (ch in 'a'..'f') {
    print(ch) // 출력: abcdef
}
```

## 2. `in` 키워드로 포함 여부 확인

`in` 키워드는 `for` 루프 외에도, 특정 값이 범위나 컬렉션에 **포함되어 있는지** 확인하는 조건식으로도 사용됩니다. 그 결과는 `Boolean`(`true` 또는 `false`)입니다.

-   `in`: 포함되어 있으면 `true`
-   `!in`: 포함되어 있지 않으면 `true`

```kotlin
val num = 7
println(num in 1..10) // true

val char = 'x'
println(char !in 'a'..'z') // false
```

## 3. 컬렉션 순회

`for` 루프는 리스트, 집합, 맵 등 모든 종류의 컬렉션을 순회하는 데 사용됩니다.

```kotlin
val languages = listOf("Kotlin", "Java", "Python")
for (lang in languages) {
    println("I love $lang")
}
```

문자열(String) 또한 문자의 컬렉션으로 취급되어 각 문자를 순회할 수 있습니다.

```kotlin
for (ch in "Kotlin") {
    print("$ch ") // 출력: K o t l i n
}
```

## 4. `repeat` 함수

단순히 특정 횟수만큼 코드를 반복하고 싶을 때는 `repeat` 함수를 사용하는 것이 더 간결합니다.

```kotlin
repeat(3) {
    println("Hi!")
}
```
> **출력:**
> Hi!
> Hi!
> Hi!
