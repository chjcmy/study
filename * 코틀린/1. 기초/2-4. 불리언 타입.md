# 코틀린 불리언 타입 (Boolean)
#Kotlin/Basic #Kotlin/TypeSystem

> [!INFO] 불리언이란?
> **불리언(Boolean)**은 논리적인 값을 나타내는 데이터 타입으로, **`true`(참)**와 **`false`(거짓)** 두 가지 값만 가질 수 있습니다. 조건문, 반복문 등 프로그램의 흐름을 제어하는 데 필수적인 역할을 합니다.

## 1. 불리언 선언

`Boolean` 타입으로 변수를 선언하고 `true` 또는 `false` 값을 할당합니다.

```kotlin
val isKotlinFun: Boolean = true
val isProgrammingHard: Boolean = false

println(isKotlinFun) // 출력: true
```

## 2. 논리 연산자

여러 불리언 값을 조합하여 더 복잡한 논리를 만들 때 사용합니다.

| 연산자 | 이름 | 설명 | 예시 | 결과 |
| :---: | :--- | :--- | :--- | :---: |
| `&&` | AND | 두 조건이 **모두** `true`일 때만 `true` | `true && false` | `false` |
| `||` | OR | 두 조건 중 **하나라도** `true`이면 `true` | `true \|\| false` | `true` |
| `!` | NOT | 값을 **반전**시킴 (`true` -> `false`, `false` -> `true`) | `!true` | `false` |

```kotlin
val isSunny = true
val isWarm = false

// 화창하고 따뜻해야만 외출
val canGoOut = isSunny && isWarm
println("Can I go out? $canGoOut") // 출력: Can I go out? false
```

## 3. 비교 연산자

두 값을 비교하고, 그 결과를 `Boolean` 타입으로 반환합니다.

| 연산자 | 설명 | 예시 | 결과 |
| :---: | :--- | :--- | :---: |
| `==` | 값이 같음 | `5 == 5` | `true` |
| `!=` | 값이 다름 | `5 != 3` | `true` |
| `>` | 왼쪽이 큼 | `5 > 3` | `true` |
| `<` | 오른쪽이 큼 | `3 < 5` | `true` |
| `>=` | 왼쪽이 크거나 같음 | `5 >= 5` | `true` |
| `<=` | 오른쪽이 크거나 같음 | `3 <= 5` | `true` |

## 4. 조건문에서의 활용

`if`나 `when`과 같은 조건문에서 분기 처리를 위해 사용됩니다.

```kotlin
val age = 20

if (age >= 19) {
    println("성인입니다.")
} else {
    println("미성년자입니다.")
}
```

불리언 값을 반환하는 함수를 만들어 사용하면 코드가 더 명확해집니다.

```kotlin
fun isAdult(age: Int): Boolean {
    return age >= 19
}

if (isAdult(20)) {
    println("성인입니다.")
}
```

## 5. 컬렉션과 함께 사용

컬렉션 라이브러리의 여러 함수들은 불리언 조건을 활용하여 데이터를 필터링하거나 검사합니다.

```kotlin
val numbers = listOf(1, 2, 3, 4, 5)

// 리스트에 짝수가 하나라도 있는지 검사
val hasEvenNumber = numbers.any { it % 2 == 0 }
println("리스트에 짝수가 있나요? $hasEvenNumber") // 출력: 리스트에 짝수가 있나요? true

// 리스트의 모든 숫자가 0보다 큰지 검사
val allPositive = numbers.all { it > 0 }
println("모든 숫자가 양수인가요? $allPositive") // 출력: 모든 숫자가 양수인가요? true
```
