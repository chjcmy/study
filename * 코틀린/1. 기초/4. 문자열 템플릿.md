# 문자열 템플릿 (String Templates)
#Kotlin/Basic

> [!INFO] 문자열 템플릿이란?
> 문자열 템플릿(또는 문자열 보간, String Interpolation)은 문자열 리터럴 안에 변수나 표현식의 값을 직접 포함시키는 기능입니다. 자바에서 `+` 연산자로 문자열을 길게 이어 붙이던 방식보다 훨씬 간결하고 가독성이 좋습니다.

## 1. 기본 사용법

### 변수 삽입 (`$`) 

변수 이름 앞에 `$` 기호를 붙이면, 해당 변수의 값이 문자열에 포함됩니다.

```kotlin
val name = "Kotlin"
println("Hello, $name!") // 출력: Hello, Kotlin!
```

### 표현식 삽입 (`${...}`)

단순한 변수가 아닌, 연산이나 함수 호출 등 복잡한 표현식의 결과를 포함시키려면 중괄호(`${...}`)로 감싸줍니다.

```kotlin
val a = 10
val b = 20
println("The sum of $a and $b is ${a + b}") // 출력: The sum of 10 and 20 is 30

val text = "Kotlin"
println("The length of '$text' is ${text.length}") // 출력: The length of 'Kotlin' is 6
```

## 2. 다중 줄 문자열 (Raw String)

세 개의 큰따옴표(`"""`)를 사용하면, 이스케이프 문자 없이 여러 줄에 걸친 문자열을 쉽게 작성할 수 있습니다. 문자열 템플릿도 그대로 사용할 수 있습니다.

```kotlin
val name = "홍길동"
val message = """
    |안녕하세요, $name 님.
    |오늘의 공지사항입니다.
    |감사합니다.
    """.trimMargin() // | 문자를 기준으로 앞의 공백을 제거

println(message)
```
> **출력:**
> 안녕하세요, 홍길동 님.
> 오늘의 공지사항입니다.
> 감사합니다.

## 3. 주의사항

### `$ ` 문자 자체를 써야 할 때

문자열에 `$` 기호 자체를 포함시키려면 이스케이프 문자(`\`)를 사용해야 합니다.

```kotlin
println("The price is \$10") // 출력: The price is $10
```

### 변수와 일반 텍스트가 붙어 있을 때

변수 이름 뒤에 바로 다른 문자가 이어져서 변수의 경계가 모호할 때는, 변수를 중괄호로 감싸주어야 합니다.

```kotlin
val price = 100
// println("Price: $priceUSD") // 컴파일 오류! 'priceUSD'라는 변수를 찾으려고 함
println("Price: ${price}USD") // 올바른 사용법. 출력: Price: 100USD
```
