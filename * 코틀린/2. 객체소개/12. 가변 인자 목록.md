# 가변 인자 (vararg)
#Kotlin/Function

> [!INFO] 가변 인자(vararg)란?
> **가변 인자(Variable-number Arguments)**는 함수를 호출할 때, **인자의 개수를 정해두지 않고 동적으로** 여러 개의 값을 전달할 수 있도록 하는 기능입니다. 코틀린에서는 `vararg` 키워드를 사용하여 이를 구현합니다.

`vararg`를 사용하면, 동일한 함수에 `sum(1, 2)`, `sum(1, 2, 3, 4)`, `sum()` 등 다양한 개수의 인자를 전달할 수 있어 코드의 유연성이 크게 향상됩니다.

## 1. 기본 사용법

함수의 파라미터 앞에 `vararg` 키워드를 붙여 선언합니다. `vararg` 파라미터는 함수 내부에서 **배열(Array)**로 취급됩니다.

```kotlin
fun printAll(vararg items: String) {
    println("--- Items ---")
    for (item in items) {
        println(item)
    }
}

fun main() {
    printAll("Apple", "Banana", "Cherry")
    printAll("Dog")
    printAll() // 인자를 전달하지 않아도 됨
}
```

## 2. 일반 파라미터와 함께 사용

`vararg`는 다른 일반 파라미터와 함께 사용할 수 있습니다.

> [!IMPORTANT] `vararg` 파라미터는 **하나만** 사용할 수 있으며, 일반적으로 **파라미터 목록의 가장 마지막**에 위치해야 합니다.

```kotlin
fun greetAll(greeting: String, vararg names: String) {
    for (name in names) {
        println("$greeting, $name!")
    }
}

fun main() {
    greetAll("Hello", "Alice", "Bob", "Charlie")
}
```

## 3. 스프레드 연산자 (`*`)

이미 배열이나 리스트에 담겨 있는 값들을 `vararg` 파라미터로 전달하고 싶을 때는 **스프레드(spread) 연산자 `*`** 를 사용해야 합니다. 이 연산자는 컬렉션의 각 원소를 '펼쳐서' 인자로 전달하는 역할을 합니다.

```kotlin
fun main() {
    val fruits = arrayOf("Apple", "Banana", "Cherry")

    // printAll(fruits) // 컴파일 오류! 타입 불일치
    printAll(*fruits) // OK. printAll("Apple", "Banana", "Cherry")와 동일
}
```

## 4. `vararg`의 장점

### 함수 오버로딩 대체

인자의 개수만 다른 여러 함수를 만들어야 하는 경우(오버로딩), `vararg`를 사용하면 하나의 함수로 간결하게 대체할 수 있습니다.

-   **오버로딩 방식**
    ```kotlin
    fun sum(a: Int, b: Int): Int = a + b
    fun sum(a: Int, b: Int, c: Int): Int = a + b + c
    ```
-   **`vararg` 방식**
    ```kotlin
    fun sum(vararg numbers: Int): Int {
        return numbers.sum()
    }

    println(sum(1, 2))       // 출력: 3
    println(sum(1, 2, 3))    // 출력: 6
    ```
