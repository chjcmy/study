# 코틀린의 리스트 (List)
#Kotlin/Collection

> [!INFO] 리스트란?
> **리스트(List)**는 코틀린의 기본 컬렉션 타입 중 하나로, **순서가 있는** 데이터의 집합을 다룹니다. 리스트의 각 요소는 인덱스(index)를 통해 접근할 수 있습니다.

코틀린의 리스트는 크게 두 종류로 나뉩니다.
-   **`List`**: 한 번 생성하면 내용을 변경할 수 없는 **불변(Immutable) 리스트**
-   **`MutableList`**: 생성 후에도 원소를 추가, 삭제, 수정할 수 있는 **가변(Mutable) 리스트**

## 1. 리스트 생성

### 불변 리스트 (`List`)
`listOf()` 함수를 사용하여 생성합니다.
```kotlin
// listOf()는 List<T> 타입을 반환
val fruits: List<String> = listOf("Apple", "Banana", "Cherry")
println(fruits) // 출력: [Apple, Banana, Cherry]

// fruits.add("Grape") // 컴파일 오류! List는 변경할 수 없음
```

### 가변 리스트 (`MutableList`)
`mutableListOf()` 함수를 사용하여 생성합니다.
```kotlin
// mutableListOf()는 MutableList<T> 타입을 반환
val numbers: MutableList<Int> = mutableListOf(1, 2, 3)
println(numbers) // 출력: [1, 2, 3]

numbers.add(4) // 원소 추가
numbers.removeAt(0) // 0번 인덱스의 원소 삭제
numbers[1] = 100 // 1번 인덱스의 원소 수정

println(numbers) // 출력: [2, 100, 4]
```

> [!TIP] 불변성을 지향하라
> 특별한 이유가 없다면, 우선적으로 불변 리스트(`List`)를 사용하는 것이 좋습니다. 데이터의 상태가 변하지 않으므로 더 안전하고 예측 가능한 코드를 작성할 수 있습니다.

## 2. 리스트의 기본 사용법

-   **요소 접근**: 인덱스 연산자 `[]`를 사용합니다.
    ```kotlin
    println(fruits[0]) // 출력: Apple
    ```
-   **크기 확인**: `size` 프로퍼티를 사용합니다.
    ```kotlin
    println(fruits.size) // 출력: 3
    ```
-   **포함 여부 확인**: `in` 키워드를 사용합니다.
    ```kotlin
    if ("Banana" in fruits) {
        println("바나나가 있습니다.")
    }
    ```
-   **반복**: `for` 루프나 `forEach` 함수를 사용합니다.
    ```kotlin
    for (fruit in fruits) {
        println(fruit)
    }
    // 또는
    fruits.forEach { println(it) }
    ```

## 3. 유용한 컬렉션 함수

코틀린은 리스트를 더 효율적으로 다룰 수 있는 다양한 확장 함수를 제공합니다.

-   **`filter`**: 조건에 맞는 원소만 골라 새로운 리스트를 만듭니다.
    ```kotlin
    val numbers = listOf(1, 2, 3, 4, 5)
    val evenNumbers = numbers.filter { it % 2 == 0 }
    println(evenNumbers) // 출력: [2, 4]
    ```
-   **`map`**: 각 원소를 특정 형태로 변환하여 새로운 리스트를 만듭니다.
    ```kotlin
    val numbers = listOf(1, 2, 3)
    val squaredNumbers = numbers.map { it * it }
    println(squaredNumbers) // 출력: [1, 4, 9]
    ```
-   **`sorted` / `sortedDescending`**: 원소를 정렬하여 새로운 리스트를 만듭니다.
    ```kotlin
    val numbers = listOf(5, 1, 4, 2, 3)
    println(numbers.sorted()) // 출력: [1, 2, 3, 4, 5]
    ```
