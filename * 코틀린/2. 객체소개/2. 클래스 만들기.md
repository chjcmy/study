# 클래스 만들기
#Kotlin/ObjectOriented

> [!INFO] 클래스(Class)란?
> **클래스(Class)**는 객체(Object)를 생성하기 위한 **설계도 또는 템플릿**입니다. 클래스에는 객체가 가질 데이터(프로퍼티)와 객체가 수행할 동작(멤버 함수)이 정의되어 있습니다.

## 1. 클래스 정의와 객체 생성

`class` 키워드를 사용하여 클래스를 정의하고, 생성자를 호출하여 객체(인스턴스)를 생성합니다.

```kotlin
// 'Dog'라는 이름의 클래스 정의
class Dog {
    // 멤버 함수 정의
    fun bark() = "Woof!"
}

fun main() {
    // Dog 클래스의 인스턴스(객체) 생성
    val myDog = Dog()

    // 객체의 멤버 함수 호출
    println(myDog.bark()) // 출력: Woof!
}
```

## 2. 프로퍼티 (Properties)

> [!NOTE]
> 프로퍼티는 클래스 내부에 `val` 또는 `var`로 선언되어 객체의 **상태(데이터)**를 저장합니다.

```kotlin
class Cup {
    // percentFull 이라는 Int 타입 프로퍼티 선언 및 초기화
    var percentFull = 0
}

fun main() {
    val myCup = Cup()

    // 점(.)을 통해 프로퍼티에 접근하여 값 변경
    myCup.percentFull = 50

    println(myCup.percentFull) // 출력: 50
}
```

## 3. 주 생성자 (Primary Constructor)

> [!IMPORTANT]
> 주 생성자는 클래스 헤더의 일부로, 객체를 생성할 때 프로퍼티를 초기화하는 가장 간결한 방법을 제공합니다.

-   생성자 파라미터를 `val` 또는 `var`로 선언하면, 해당 파라미터는 즉시 클래스의 프로퍼티가 됩니다.

```kotlin
// 클래스를 선언함과 동시에 name과 age 프로퍼티를 갖는 주 생성자를 정의
class Person(
    val name: String, // 읽기 전용 프로퍼티
    var age: Int      // 변경 가능 프로퍼티
)

fun main() {
    // 생성자를 통해 프로퍼티 값을 초기화하며 객체 생성
    val person = Person("Alice", 25)

    println(person.name) // 출력: Alice
    println(person.age)  // 출력: 25
}
```

## 4. 멤버 함수 (Member Functions)

> [!NOTE]
> 멤버 함수는 클래스 내부에 정의된 함수로, 객체의 **동작(Behavior)**을 나타냅니다. 객체의 프로퍼티를 사용하거나 특정 로직을 수행합니다.

```kotlin
class Calculator {
    fun add(a: Int, b: Int): Int {
        return a + b
    }
}

fun main() {
    val calc = Calculator()
    val result = calc.add(3, 5)
    println(result) // 출력: 8
}
```

## 5. `this` 키워드

`this`는 클래스 내부에서 **현재 객체(자기 자신)의 인스턴스**를 가리킵니다. 프로퍼티와 함수의 이름이 중복될 때 멤버임을 명확히 하거나, 다른 함수에 자기 자신을 전달할 때 사용됩니다.

```kotlin
class Hamster {
    fun speak() = "Squeak!"
    fun exercise() = this.speak() + " Running on wheel" // this는 Hamster 객체 자신
}
```
