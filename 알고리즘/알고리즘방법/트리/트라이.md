# 트라이 (Trie)

## 1. 개요
문자열을 빠르게 검색하기 위한 트리 형태의 자료구조입니다.
**접두사 트리 (Prefix Tree)** 라고도 불립니다. 검색어 자동 완성, 사전 찾기 등에 활용됩니다.

## 2. 특징
- 문자열 길이(L)에 비례하는 시간 복잡도 $O(L)$ 로 검색이 가능합니다.
- 많은 메모리를 사용한다는 단점이 있습니다.

## 3. 파이썬 구현

```python
class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end_of_word = False

class Trie:
    def __init__(self):
        self.root = TrieNode()

    # 문자열 삽입
    def insert(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]
        node.is_end_of_word = True

    # 문자열 검색
    def search(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                return False
            node = node.children[char]
        return node.is_end_of_word

    # 접두사 확인 (StartsWith)
    def starts_with(self, prefix):
        node = self.root
        for char in prefix:
            if char not in node.children:
                return False
            node = node.children[char]
        return True

# 사용 예시
trie = Trie()
trie.insert("apple")
trie.insert("app")

print(f"Search 'apple': {trie.search('apple')}") # True
print(f"Search 'app': {trie.search('app')}")     # True
print(f"Search 'ap': {trie.search('ap')}")       # False (단어가 아님)
print(f"Starts with 'ap': {trie.starts_with('ap')}") # True
```
