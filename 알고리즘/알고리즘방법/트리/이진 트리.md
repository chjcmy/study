# 이진 트리 (Binary Tree)

## 1. 개요
모든 노드가 **최대 2개의 자식** (왼쪽, 오른쪽)을 가지는 트리입니다.
가장 많이 사용되는 트리 형태이며, 이진 탐색 트리(BST), 힙(Heap) 등의 기반이 됩니다.

## 2. 이진 트리의 종류
- **포화 이진 트리 (Perfect Binary Tree)**: 모든 레벨이 꽉 차 있음
- **완전 이진 트리 (Complete Binary Tree)**: 위에서 아래로, 왼쪽부터 차곡차곡 채워진 트리
- **편향 이진 트리 (Skewed Binary Tree)**: 한쪽으로만 노드가 이어진 경우 (사실상 연결 리스트)

## 3. 순회 방식 (Traversal)
트리의 모든 노드를 방문하는 방법입니다.
1.  **전위 순회 (Pre-order)**: Root -> Left -> Right
2.  **중위 순회 (In-order)**: Left -> Root -> Right
3.  **후위 순회 (Post-order)**: Left -> Right -> Root

## 4. 파이썬 구현

```python
class Node:
    def __init__(self, data):
        self.data = data
        self.left = None
        self.right = None

# 전위 순회 (Pre-order)
def preorder(node):
    if node:
        print(node.data, end=' ')
        preorder(node.left)
        preorder(node.right)

# 중위 순회 (In-order)
def inorder(node):
    if node:
        inorder(node.left)
        print(node.data, end=' ')
        inorder(node.right)

# 후위 순회 (Post-order)
def postorder(node):
    if node:
        postorder(node.left)
        postorder(node.right)
        print(node.data, end=' ')

# 트리 생성 및 테스트
#      1
#     / \
#    2   3
#   / \
#  4   5

root = Node(1)
root.left = Node(2)
root.right = Node(3)
root.left.left = Node(4)
root.left.right = Node(5)

print("Pre-order:", end=' ')
preorder(root) # 1 2 4 5 3
print("\nIn-order:", end=' ')
inorder(root)  # 4 2 5 1 3
print("\nPost-order:", end=' ')
postorder(root) # 4 5 2 3 1
```
