# 트리 알아보기 (Tree Basics)

## 1. 개요
트리는 **노드(Node)** 와 **간선(Edge)** 으로 이루어진 계층적(Hierarchical) 자료구조입니다.
사이클(Cycle)이 없는 연결 그래프이며, 그래프의 특수한 형태입니다.

## 2. 주요 용어
- **루트 노드 (Root)**: 트리의 가장 위에 있는 시작 노드 (부모가 없음)
- **단말 노드 (Leaf)**: 자식 노드가 없는 노드 (말단 노드)
- **부모 (Parent) & 자식 (Child)**: 연결된 두 노드 중 상위가 부모, 하위가 자식
- **깊이 (Depth)**: 루트에서 특정 노드까지 도달하는 데 거치는 간선의 수
- **높이 (Height)**: 트리에서 가장 깊은 노드의 깊이

## 3. 트리의 특징
1.  **계층 구조**: 데이터 간의 상하 관계를 표현 (예: 파일 시스템, 조직도)
2.  **비순환**: 사이클이 존재하지 않음 (루트에서 어떤 노드로 가는 경로는 유일함)
3.  **N-1 간선**: N개의 노드가 있다면 간선은 항상 N-1개 존재

## 4. 파이썬 구현 (기본 구조 & 순회)

```python
class TreeNode:
    def __init__(self, value):
        self.value = value
        self.children = []

    def add_child(self, child_node):
        self.children.append(child_node)

def print_tree(node, level=0):
    indent = "  " * level
    print(f"{indent}- {node.value}")
    for child in node.children:
        print_tree(child, level + 1)

# 사용 예시
root = TreeNode("Root")
child1 = TreeNode("Child 1")
child2 = TreeNode("Child 2")
root.add_child(child1)
root.add_child(child2)
child1.add_child(TreeNode("Grandchild 1.1"))

print_tree(root)
# - Root
#   - Child 1
#     - Grandchild 1.1
#   - Child 2
```
