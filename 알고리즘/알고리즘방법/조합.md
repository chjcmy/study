# 조합 (Combinatorics)

> [!INFO] 상세 내용
> [조합 알아보기](조합/조합%20알아보기.md)

## 1. 개요
서로 다른 n개 중에서 r개를 선택하는 경우의 수를 구하는 것입니다. (순서는 고려하지 않음)
순서를 고려하면 **순열(Permutation)** 입니다.

## 2. 파이썬 구현
파이썬 `itertools` 라이브러리를 사용하면 편리합니다.

```python
from itertools import combinations, permutations

data = ['A', 'B', 'C']

# 조합 (순서 X)
result_comb = list(combinations(data, 2))
print(f"조합: {result_comb}") 
# [('A', 'B'), ('A', 'C'), ('B', 'C')]

# 순열 (순서 O)
result_perm = list(permutations(data, 2))
print(f"순열: {result_perm}")
# [('A', 'B'), ('A', 'C'), ('B', 'A'), ('B', 'C'), ('C', 'A'), ('C', 'B')]
```
