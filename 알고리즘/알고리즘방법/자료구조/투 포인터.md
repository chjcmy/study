# 투 포인터 (Two Pointers)

## 1. 개요
배열이나 리스트에서 **두 개의 포인터(인덱스)** 를 사용하여 데이터를 탐색하고 처리하는 알고리즘입니다. 주로 정렬된 배열에서 두 수의 합을 찾거나, 특정 조건을 만족하는 연속 부분 배열을 찾을 때 사용합니다.

## 2. 동작 과정 (예: 정렬된 배열에서 두 수의 합 찾기)
1. **시작 포인터(start)** 는 배열의 맨 앞(0)에, **끝 포인터(end)** 는 배열의 맨 뒤(n-1)에 위치시킵니다.
2. 두 포인터가 가리키는 값의 합을 계산합니다.
3. 합과 목표값(Target)을 비교하여 포인터를 이동합니다.
    - **합 > 타겟**: 합을 줄여야 하므로 `end` 포인터를 왼쪽으로 이동 (`end -= 1`)
    - **합 < 타겟**: 합을 늘려야 하므로 `start` 포인터를 오른쪽으로 이동 (`start += 1`)
    - **합 == 타겟**: 목표를 찾았으므로 종료하거나 카운트 증가.
4. 두 포인터가 교차할 때까지 반복합니다.

## 3. 시간 복잡도
- **O(n)**: 배열을 한 번만 순회하므로 O(n²)에 비해 매우 효율적입니다.

## 4. 파이썬(Python) 구현 코드

```python
def two_pointer_sum(arr, target):
    # 투 포인터는 주로 정렬된 배열에서 사용됨
    arr.sort() 
    
    start = 0
    end = len(arr) - 1
    count = 0
    
    while start < end:
        current_sum = arr[start] + arr[end]
        
        if current_sum == target:
            count += 1
            print(f"Found: {arr[start]} + {arr[end]} = {target}")
            start += 1
            end -= 1
        elif current_sum < target:
            start += 1
        else: # current_sum > target
            end -= 1
            
    return count

# 사용 예시
data = [2, 7, 11, 15, 1, 8, 3] # 정렬 후: [1, 2, 3, 7, 8, 11, 15]
target_sum = 9
two_pointer_sum(data, target_sum)
# 1+8=9, 2+7=9
```
