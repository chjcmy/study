# 기하 알아보기 (Geometry)

## 1. 개요
기하학은 점, 선, 면 등 도형의 성질을 다루는 분야입니다.
코딩 테스트에서는 주로 좌표 평면 위에서 점들의 관계를 파악하는 문제(CCW, 선분 교차, 볼록 껍질 등)가 출제됩니다.

## 2. CCW (Counter Clockwise)
**CCW**는 세 점 A, B, C가 주어졌을 때, 이 점들의 방향 관계를 파악하는 가장 기초적이고 중요한 알고리즘입니다.
세 점의 순서대로 화살표를 그렸을 때의 방향을 의미합니다.

### 2.1 판별 공식 (외적 활용)
세 점 $P1(x1, y1)$, $P2(x2, y2)$, $P3(x3, y3)$가 있을 때:
$$ (x2 - x1)(y3 - y1) - (x3 - x1)(y2 - y1) $$

### 2.2 결과의 의미
- **양수 (+)**: 반시계 방향 (Counter Clockwise)
- **음수 (-)**: 시계 방향 (Clockwise)
- **0**: 일직선 상에 있음 (Collinear)

## 3. 파이썬 구현 코드

```python
def ccw(p1, p2, p3):
    """
    세 점 p1, p2, p3의 방향 관계를 반환
    :return: 1 (반시계), -1 (시계), 0 (일직선)
    """
    x1, y1 = p1
    x2, y2 = p2
    x3, y3 = p3
    
    # 외적 계산
    result = (x2 - x1) * (y3 - y1) - (x3 - x1) * (y2 - y1)
    
    if result > 0:
        return 1
    elif result < 0:
        return -1
    else:
        return 0

# 사용 예시
point_a = (1, 1)
point_b = (5, 5)
point_c = (7, 3)

res = ccw(point_a, point_b, point_c)
if res == 1:
    print("반시계 방향")
elif res == -1:
    print("시계 방향")
else:
    print("일직선")
```

## 4. 기타 기하 알고리즘
- **선분 교차 판별**: 두 선분이 교차하는지 CCW를 응용하여 확인
- **볼록 껍질 (Convex Hull)**: 여러 점들을 모두 포함하는 가장 작은 볼록 다각형 찾기 (그라함 스캔 등)
