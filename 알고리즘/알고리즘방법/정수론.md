# 정수론 (Number Theory)

## 1. 개요
수의 성질을 탐구하는 분야로, 코딩 테스트에서는 주로 **소수(Prime Number)** 판별, **최대공약수(GCD)**, **최소공배수(LCM)**, **에라토스테네스의 체** 등이 자주 출제됩니다.

## 2. 주요 알고리즘

### 2.1 최대공약수 (GCD) & 최소공배수 (LCM)
- **유클리드 호제법 (Euclidean Algorithm)** 을 사용하면 효율적으로 구할 수 있습니다.
- $LCM(a, b) = (a \times b) / GCD(a, b)$

### 2.2 소수 판별
- **기본 판별**: 2부터 $\sqrt{N}$ 까지 나누어 떨어지는지 확인 ($O(\sqrt{N})$)
- **에라토스테네스의 체**: 범위 내의 모든 소수를 빠르게 구할 때 사용 ($O(N \log \log N)$)

## 3. 파이썬(Python) 구현 코드

```python
import math

# 1. 최대공약수 (GCD)
def gcd(a, b):
    while b > 0:
        a, b = b, a % b
    return a

# 2. 최소공배수 (LCM)
def lcm(a, b):
    return a * b // gcd(a, b)

# 3. 소수 판별 (단일 숫자)
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(math.sqrt(n)) + 1):
        if n % i == 0:
            return False
    return True

# 4. 에라토스테네스의 체 (범위 내 소수 구하기)
def sieve_of_eratosthenes(limit):
    sieve = [True] * (limit + 1)
    sieve[0] = sieve[1] = False
    
    for i in range(2, int(math.sqrt(limit)) + 1):
        if sieve[i]:
            for j in range(i * i, limit + 1, i):
                sieve[j] = False
                
    return [x for x in range(limit + 1) if sieve[x]]

# 사용 예시
print(f"GCD(12, 18): {gcd(12, 18)}")
print(f"LCM(12, 18): {lcm(12, 18)}")
print(f"Is 17 prime? {is_prime(17)}")
print(f"Primes upto 30: {sieve_of_eratosthenes(30)}")
```
