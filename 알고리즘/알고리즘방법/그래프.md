# 그래프 (Graph)

## 1. 개요
**정점(Vertex)** 과 **간선(Edge)** 으로 이루어진 자료구조입니다.
코딩 테스트에서는 주로 **탐색(DFS/BFS)**, **최단 경로**, **최소 신장 트리(MST)** 등의 문제가 출제됩니다.

## 2. 주요 알고리즘
- **DFS/BFS**: 그래프의 모든 정점을 탐색
- **유니온 파인드**: 서로소 집합(Disjoint Set) 관리
- **위상 정렬**: 순서가 있는 작업을 정렬
- **최단 경로**: 다익스트라(한 정점), 플로이드-워셜(모든 정점)
- **최소 신장 트리 (MST)**: 크루스칼, 프림

## 3. 파이썬 코드 (DFS & BFS)

```python
from collections import deque

# 그래프 인접 리스트 표현
graph = {
    1: [2, 3, 8],
    2: [1, 7],
    3: [1, 4, 5],
    4: [3, 5],
    5: [3, 4],
    6: [7],
    7: [2, 6, 8],
    8: [1, 7]
}

# DFS (재귀)
def dfs(v, visited):
    visited[v] = True
    print(v, end=' ')
    for i in graph[v]:
        if not visited[i]:
            dfs(i, visited)

# BFS (큐)
def bfs(start, visited):
    queue = deque([start])
    visited[start] = True
    while queue:
        v = queue.popleft()
        print(v, end=' ')
        for i in graph[v]:
            if not visited[i]:
                queue.append(i)
                visited[i] = True

print("DFS:", end=' ')
dfs(1, [False] * 9)

print("\nBFS:", end=' ')
bfs(1, [False] * 9)
```
