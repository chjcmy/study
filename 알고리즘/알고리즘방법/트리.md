# 트리 (Tree)

## 1. 개요
계층적인 자료를 표현하는 비선형 자료구조입니다. 노드(Node)와 간선(Edge)으로 이루어져 있으며, 사이클이 없는 그래프의 일종입니다.

## 2. 주요 개념
- **루트 노드 (Root)**: 트리의 최상위 노드
- **단말 노드 (Leaf)**: 자식이 없는 노드
- **이진 트리 (Binary Tree)**: 모든 노드의 자식이 최대 2개인 트리
- **세그먼트 트리 (Segment Tree)**: 구간 합 등을 빠르게 구하기 위한 트리 등의 특수한 형태도 존재합니다.
