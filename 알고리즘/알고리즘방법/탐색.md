# 탐색 (Search)

## 1. 개요
데이터 집합에서 원하는 데이터를 찾는 과정입니다. 코딩 테스트에서는 주로 그래프 탐색(DFS/BFS)과 효율적인 탐색(이진 탐색)이 중요합니다.

## 2. 주요 탐색 알고리즘
- **DFS (깊이 우선 탐색)**: 깊은 부분을 우선적으로 탐색합니다. 스택이나 재귀 함수를 이용합니다.
- **BFS (너비 우선 탐색)**: 인접한 노드부터 탐색합니다. 큐(Queue)를 이용하며, 최단 경로 문제에서 유리합니다.
- **이진 탐색 (Binary Search)**: **정렬된 데이터**에서 탐색 범위를 절반씩 줄여가며 데이터를 찾습니다. 시간 복잡도는 $O(\log N)$입니다.
