# 삽입 정렬 (Insertion Sort)

## 1. 개요
데이터를 하나씩 확인하며, **이미 정렬된 부분 집합(배열의 앞부분)** 내에서 적절한 위치를 찾아 삽입하는 방식입니다.

## 2. 동작 과정
1. 두 번째 데이터부터 시작합니다. (첫 번째 데이터는 이미 정렬된 상태로 간주)
2. 현재 선택된 데이터를 앞쪽의 이미 정렬된 데이터들과 비교하여, 들어갈 위치를 찾습니다.
3. 적절한 위치에 데이터를 삽입하고, 나머지 데이터들은 뒤로 미룹니다.

## 3. 시간 복잡도
- **최악의 경우**: O(n²) (역순 정렬된 경우)
- **최선의 경우**: O(n) (이미 정렬된 경우, 비교만 하고 이동 없음)
- **평균**: O(n²)

## 4. 장단점
- **장점**: 데이터가 거의 정렬되어 있을 때 매우 효율적입니다. 안정한 정렬(Stable Sort)입니다.
- **단점**: 데이터 이동이 많아 배열의 크기가 클 경우 비효율적입니다.

## 5. 파이썬(Python) 구현 코드

```python
def insertion_sort(arr):
    n = len(arr)
    # 두 번째 원소(인덱스 1)부터 시작
    for i in range(1, n):
        key = arr[i] # 삽입할 데이터
        j = i - 1
        
        # key 값보다 큰 원소들을 한 칸씩 뒤로 밀기
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        
        # 빈 자리에 key 삽입
        arr[j + 1] = key
    return arr

# 사용 예시
data = [12, 11, 13, 5, 6]
print(f"정렬 전: {data}")
sorted_data = insertion_sort(data)
print(f"정렬 후: {sorted_data}")
```
