# 선택 정렬 (Selection Sort)

## 1. 개요
전체 데이터 중에서 **가장 작은(또는 가장 큰) 데이터를 선택**해 맨 앞(또는 정렬되지 않은 영역의 맨 앞)의 데이터와 교환하는 방식입니다.

## 2. 동작 과정 (오름차순 기준)
1. 주어진 리스트 전체를 순회하여 **최솟값**을 찾습니다.
2. 최솟값을 리스트의 맨 앞(첫 번째 위치)에 있는 값과 교환합니다.
3. 맨 앞의 위치를 제외한 나머지 리스트를 대상으로 위 과정을 반복합니다.

## 3. 시간 복잡도
- **최악, 최선, 평균** 모두 **O(n²)** 입니다. (비교 횟수가 항상 일정함)

## 4. 장단점
- **장점**: 구현이 간단합니다. 비교 횟수는 많지만 교환(Swap) 횟수는 적습니다.
- **단점**: 시간 복잡도가 O(n²)로 비효율적입니다. 불안정 정렬(Unstable Sort)입니다.

## 5. 파이썬(Python) 구현 코드

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n - 1):
        # 현재 위치(i)부터 끝까지 중에서 최솟값의 인덱스 찾기
        min_idx = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        
        # 최솟값과 현재 위치의 값 교환 (Swap)
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
    return arr

# 사용 예시
data = [64, 25, 12, 22, 11]
print(f"정렬 전: {data}")
sorted_data = selection_sort(data)
print(f"정렬 후: {sorted_data}")
```
