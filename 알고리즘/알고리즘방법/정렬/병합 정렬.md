# 병합 정렬 (Merge Sort)

## 1. 개요
리스트를 절반으로 잘라 비슷한 크기의 두 부분 리스트로 나눈 뒤, 각 부분 리스트를 재귀적으로 정렬하고, 다시 하나의 정렬된 리스트로 **합병(Merge)** 하는 분할 정복(Divide and Conquer) 방식입니다.

## 2. 동작 과정
1. **분할(Divide)**: 리스트를 절반으로 나눕니다. 크기가 1이 될 때까지 반복합니다.
2. **정복(Conquer)**: 각 부분 리스트를 재귀적으로 정렬합니다.
3. **결합(Combine)**: 두 부분 리스트를 다시 하나의 정렬된 리스트로 합칩니다. 이때 값을 비교하여 순서대로 넣습니다.

## 3. 시간 복잡도
- **항상**: **O(n log n)** (최악, 최선, 평균 모두 동일)

## 4. 장단점
- **장점**: 입력 데이터의 분포에 상관없이 일정한 성능을 보장합니다. 안정 정렬(Stable Sort)입니다.
- **단점**: 임시 배열을 위한 추가적인 메모리 공간이 필요합니다.

## 5. 파이썬(Python) 구현 코드

```python
def merge_sort(arr):
    if len(arr) < 2:
        return arr

    mid = len(arr) // 2
    low_arr = merge_sort(arr[:mid])
    high_arr = merge_sort(arr[mid:])

    merged_arr = []
    l = h = 0
    while l < len(low_arr) and h < len(high_arr):
        if low_arr[l] < high_arr[h]:
            merged_arr.append(low_arr[l])
            l += 1
        else:
            merged_arr.append(high_arr[h])
            h += 1
    
    # 남은 요소들 추가
    merged_arr += low_arr[l:]
    merged_arr += high_arr[h:]
    return merged_arr

# 사용 예시
data = [6, 5, 3, 1, 8, 7, 2, 4]
print(f"정렬 전: {data}")
sorted_data = merge_sort(data)
print(f"정렬 후: {sorted_data}")
```
