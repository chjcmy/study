## 1. 개요
기하학적 도형(점, 선, 다각형 등)을 다루는 알고리즘입니다. 코딩 테스트에서는 주로 **CCW (Counter Clockwise)**, **볼록 껍질 (Convex Hull)** 등이 출제됩니다.

## 2. CCW (세 점의 방향성)
세 점 A, B, C가 주어졌을 때, 반시계 방향인지, 시계 방향인지, 일직선인지 판별하는 알고리즘입니다.
- 결과 > 0: 반시계 방향
- 결과 < 0: 시계 방향
- 결과 = 0: 일직선

## 3. 파이썬 코드

```python
def ccw(p1, p2, p3):
    # 외적(Cross Product) 활용
    # (x2-x1)(y3-y1) - (x3-x1)(y2-y1)
    val = (p2[0] - p1[0]) * (p3[1] - p1[1]) - (p3[0] - p1[0]) * (p2[1] - p1[1])
    
    if val > 0:
        return 1  # 반시계
    elif val < 0:
        return -1 # 시계
    else:
        return 0  # 일직선

p1 = (1, 1)
p2 = (5, 5)
p3 = (7, 3)
print(f"방향: {ccw(p1, p2, p3)}")
```
