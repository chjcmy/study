## 1. 개요
복잡한 문제를 **작은 부분 문제**로 나누어 해결하고, 그 결과를 **저장(Memoization)** 하여 다시 계산하지 않도록 하는 알고리즘입니다.

## 2. 핵심 조건
1. **최적 부분 구조 (Optimal Substructure)**: 큰 문제의 최적해가 작은 문제의 최적해를 포함.
2. **중복되는 부분 문제 (Overlapping Subproblems)**: 동일한 작은 문제가 반복적으로 등장.

## 3. 파이썬 코드 (피보나치 수열)
```python
# 탑다운 (Memoization)
d = [0] * 100

def fibo(x):
    if x == 1 or x == 2:
        return 1
    if d[x] != 0:
        return d[x]
    d[x] = fibo(x - 1) + fibo(x - 2)
    return d[x]

print(fibo(10))
```
