# 그리디 (Greedy)

## 1. 개요
**탐욕 알고리즘**이라고도 하며, 매 순간 **가장 좋아 보이는(최적의) 선택**을 하여 최종적인 해답에 도달하는 방법입니다. 순간의 선택은 최적이지만, 전체결과가 최적이라는 보장은 없습니다. 하지만 코딩 테스트의 많은 문제에서 최적해를 보장하거, 그리디로 풀 수 있는 문제가 자주 출제됩니다.

## 2. 예제: 거스름돈
- 가장 큰 화폐 단위부터 거슬러 주는 것이 동전 개수를 줄이는 최적의 방법입니다.

## 3. 파이썬 코드

```python
def min_coin_count(n, coins):
    count = 0
    # 큰 화폐 단위부터 확인
    for coin in coins:
        count += n // coin # 해당 화폐로 거슬러 줄 수 있는 개수
        n %= coin # 남은 금액
    return count

coins = [500, 100, 50, 10]
n = 1260
print(f"최소 동전 개수: {min_coin_count(n, coins)}")
```
