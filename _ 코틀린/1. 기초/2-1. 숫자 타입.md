# 코틀린 숫자 타입 (Number Types)
#Kotlin/Basic #Kotlin/TypeSystem

> [!INFO]
> 코틀린의 모든 데이터 타입은 객체(Object)입니다. 따라서 숫자 역시 객체로 다루어지며, 다양한 변환 함수와 프로퍼티를 가집니다. 수 타입은 크게 **정수**와 **부동소수점**으로 나뉩니다.

## 1. 정수 타입 (Integer Types)

메모리 크기와 표현 가능한 값의 범위에 따라 네 가지 타입으로 나뉩니다.

| 타입 | 크기 (Bits) | 최솟값 | 최댓값 |
| :--- | :---: | :--- | :--- |
| `Byte` | 8 | -128 | 127 |
| `Short`| 16 | -32,768 | 32,767 |
| `Int` | 32 | -2,147,483,648 (-2³¹) | 2,147,483,647 (2³¹-1) |
| `Long` | 64 | -9,223,372,036,854,775,808 (-2⁶³) | 9,223,372,036,854,775,807 (2⁶³-1) |

-   **리터럴(Literal) 표현**:
    -   일반적인 숫자는 `Int`로 추론됩니다.
    -   `Long` 타입의 값에는 숫자 뒤에 접미사 `L`을 붙입니다.
    -   16진수는 `0x`, 2진수는 `0b` 접두사를 사용합니다.

```kotlin
val intValue = 100       // Int로 추론
val longValue = 100L     // Long
val hexValue = 0x1F      // 16진수 (31)
val binaryValue = 0b1010 // 2진수 (10)
```

## 2. 부동소수점 타입 (Floating-Point Types)

소수점을 포함하는 실수를 다루며, 정밀도에 따라 두 가지 타입으로 나뉩니다.

| 타입 | 크기 (Bits) | 유효 숫자 (정밀도) |
| :--- | :---: | :--- |
| `Float` | 32 | 6-7 자리 |
| `Double`| 64 | 15-17 자리 |

-   **리터럴(Literal) 표현**:
    -   일반적인 소수점 숫자는 `Double`로 추론됩니다.
    -   `Float` 타입의 값에는 숫자 뒤에 접미사 `f` 또는 `F`를 붙입니다.

```kotlin
val doubleValue = 3.141592 // Double로 추론
val floatValue = 3.141592f // Float
```

## 3. 타입 변환 (Type Conversion)

> [!IMPORTANT] 코틀린은 자동 형 변환을 지원하지 않는다
> 코틀린에서는 작은 타입의 숫자를 큰 타입의 변수에 직접 할당할 수 없습니다. 이는 타입 불일치로 인한 오류를 방지하기 위한 설계 철학입니다. 모든 타입 변환은 반드시 **명시적으로** 이루어져야 합니다.

-   **변환 함수**: 각 숫자 타입은 다른 타입으로 변환할 수 있는 함수를 제공합니다.
    -   `toByte()`, `toShort()`, `toInt()`, `toLong()`, `toFloat()`, `toDouble()`, `toChar()`

```kotlin
val i: Int = 100
// val l: Long = i // 컴파일 오류!
val l: Long = i.toLong() // OK

val d: Double = i.toDouble() // OK
```

## 4. 연산

-   **산술 연산**: `+`, `-`, `*`, `/`, `%` 등 기본적인 산술 연산을 지원합니다.
-   **비트 연산**: `shl`(left shift), `shr`(right shift), `ushr`(unsigned right shift), `and`, `or`, `xor`, `inv`(invert) 등의 함수 형태로 제공됩니다.
    ```kotlin
    val result = 1 shl 2 // 1을 왼쪽으로 2비트 시프트 (결과: 4)
    ```
