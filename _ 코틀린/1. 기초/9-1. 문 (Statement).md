# 문 (Statement)
#Kotlin/Basic #Kotlin/Concept

> [!INFO]
> 프로그래밍 언어의 구성 요소를 이해하는 데 있어 '식(Expression)'과 '문(Statement)'의 차이를 아는 것은 매우 중요합니다. 코틀린은 **식이 중심이 되는(Expression-oriented)** 언어입니다.

## 1. 문 (Statement)

-   **정의**: 어떤 동작을 수행하도록 지시하는 코드 단위이지만, **결과 값을 만들어내지 않습니다.**
-   **목적**: 주로 프로그램의 상태를 변경하는 **부수 효과(Side Effect)**를 일으키기 위해 사용됩니다.
-   **예시**:
    -   변수 선언 (`val a = 10`)
    -   `println("Hello")` 호출
    -   자바의 `if`, `for` 등 대부분의 제어문

```java
// 자바의 예시
int result;
if (a > b) {
    result = a; // if문은 값을 반환하지 않으므로, 외부 변수에 값을 할당해야 함
} else {
    result = b;
}
```
