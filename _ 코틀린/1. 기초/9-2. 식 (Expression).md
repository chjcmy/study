# 식 (Expression)
#Kotlin/Basic #Kotlin/Concept

## 2. 식 (Expression)

-   **정의**: 평가(evaluate)된 후 **항상 결과 값을 만들어내는** 코드 단위입니다.
-   **목적**: 값을 계산하고 생성하는 것입니다. 식의 결과는 다른 변수에 할당되거나, 다른 식의 일부가 될 수 있습니다.
-   **예시**:
    -   `1 + 2`
    -   `"Hello"`
    -   `myFunction()` 호출
