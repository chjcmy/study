# break, continue, 그리고 레이블
#Kotlin/ControlFlow

> [!INFO]
> `break`, `continue`, 그리고 **레이블(Label)**은 `for`, `while`과 같은 반복문의 흐름을 정교하게 제어하는 데 사용되는 키워드입니다.

## 1. `break`

-   **역할**: `break`가 호출되는 순간, 자신을 감싸고 있는 **가장 가까운 반복문을 즉시 종료**하고 빠져나옵니다.
-   **사용처**: 특정 조건을 만족했을 때 더 이상 반복을 진행할 필요가 없을 때 사용합니다.

```kotlin
for (i in 1..10) {
    if (i == 5) {
        break // i가 5이면 for 루프를 즉시 중단
    }
    print("$i ")
}
// 출력: 1 2 3 4
```

## 2. `continue`

-   **역할**: `continue`가 호출되는 순간, 현재 반복문의 **남은 코드를 건너뛰고 즉시 다음 반복으로** 넘어갑니다.
-   **사용처**: 특정 조건에서는 이번 반복을 건너뛰고 다음 반복을 계속하고 싶을 때 사용합니다.

```kotlin
for (i in 1..5) {
    if (i == 3) {
        continue // i가 3이면 현재 반복을 건너뛰고 다음 반복(i=4)으로 진행
    }
    print("$i ")
}
// 출력: 1 2 4 5
```

## 3. 레이블 (Label)

> [!IMPORTANT] 레이블이란?
> 기본적으로 `break`와 `continue`는 가장 안쪽의 루프에만 영향을 줍니다. 중첩된 루프에서 **바깥쪽 루프를 제어**하고 싶을 때는 **레이블(Label)**을 사용해야 합니다.
>
> 레이블은 특정 루프 앞에 `이름@` 형태로 붙여서 사용합니다.

### 레이블과 `break`

`break@[레이블 이름]`을 사용하면, 지정된 레이블이 붙은 루프까지 한 번에 종료할 수 있습니다.

```kotlin
fun main() {
    outer@ for (i in 1..3) {
        for (j in 1..3) {
            if (i == 2 && j == 2) {
                println("\nBreaking the outer loop!")
                break@outer // outer 레이블이 붙은 바깥쪽 for문 전체를 종료
            }
            print("($i, $j) ")
        }
    }
}
```
> **출력:**
> (1, 1) (1, 2) (1, 3) (2, 1) 
> Breaking the outer loop!

### 레이블과 `continue`

`continue@[레이블 이름]`을 사용하면, 지정된 레이블이 붙은 루프의 다음 반복으로 바로 넘어갈 수 있습니다.

```kotlin
fun main() {
    outer@ for (i in 1..3) {
        for (j in 1..3) {
            if (j == 2) {
                println(" | Continuing to next 'i'")
                continue@outer // j가 2일 때, 바깥쪽 루프의 다음 반복(i)으로 넘어감
            }
            print("($i, $j) ")
        }
    }
}
```
> **출력:**
> (1, 1) | Continuing to next 'i'
> (2, 1) | Continuing to next 'i'
> (3, 1)

> [!WARNING] 과도한 사용 주의
> 레이블과 `goto`처럼 동작하는 `break`, `continue`는 코드의 흐름을 복잡하게 만들어 가독성을 해칠 수 있습니다. 꼭 필요한 경우에만 신중하게 사용하고, 로직이 너무 복잡해진다면 함수로 분리하는 것을 고려하는 것이 좋습니다.
