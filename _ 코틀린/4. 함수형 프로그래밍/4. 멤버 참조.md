# 멤버 및 함수 참조 (::)
#Kotlin/Functional

> [!INFO] 멤버 참조란?
> **멤버 참조(Member Reference)**는 클래스의 프로퍼티나 함수를 직접 호출하지 않고, **값으로 참조**하여 다른 함수에 인자로 전달하거나 변수에 저장할 수 있게 하는 기능입니다.
>
> `::` 연산자를 사용하여 표현하며, 람다를 더 간결하고 명확하게 표현하는 데 사용됩니다.

## 1. 함수 참조 (Function References)

이미 선언된 함수(최상위 함수, 멤버 함수, 확장 함수)를 람다 대신 값으로 전달할 수 있습니다.

### 최상위 함수 참조
```kotlin
fun isEven(n: Int): Boolean = n % 2 == 0

fun main() {
    val numbers = listOf(1, 2, 3, 4)

    // 람다를 사용하는 경우
    println(numbers.filter { isEven(it) })

    // 함수 참조를 사용하는 경우 -> 더 간결함
    println(numbers.filter(::isEven))
}
```

### 멤버 함수 참조
```kotlin
class Person(val name: String, val age: Int) {
    fun isAdult(): Boolean = age >= 19
}

fun main() {
    val people = listOf(Person("Alice", 25), Person("Bob", 17))

    // 멤버 함수 참조를 사용하여 각 Person 객체의 isAdult()를 호출
    println(people.filter(Person::isAdult))
}
```

## 2. 프로퍼티 참조 (Property References)

클래스의 프로퍼티도 `::` 연산자를 통해 값으로 참조할 수 있습니다. 이는 `map`과 같은 함수에서 특정 프로퍼티 값만 추출할 때 매우 유용합니다.

```kotlin
data class Person(val name: String, val age: Int)

fun main() {
    val people = listOf(Person("Alice", 25), Person("Bob", 20))

    // 람다를 사용하여 이름만 추출
    println(people.map { it.name })

    // 프로퍼티 참조를 사용하여 이름만 추출
    println(people.map(Person::name))
}
```

## 3. 생성자 참조 (Constructor References)

생성자 또한 `::` 연산자를 통해 참조할 수 있습니다. 이는 컬렉션의 각 원소를 받아 객체를 생성할 때 유용합니다.

```kotlin
data class User(val name: String)

fun main() {
    val names = listOf("Alice", "Bob", "Charlie")

    // 각 이름을 사용하여 User 객체를 생성
    val users = names.map(::User)
    println(users) // 출력: [User(name=Alice), User(name=Bob), User(name=Charlie)]
}
```

## 4. 바운드 멤버 참조 (Bound Member References)

> [!NOTE]
> 특정 객체의 인스턴스에 대한 멤버를 참조하는 것을 **바운드(Bound) 멤버 참조**라고 합니다. 이 경우, 수신 객체가 이미 참조에 '묶여' 있으므로, 함수 호출 시 수신 객체를 전달할 필요가 없습니다.

```kotlin
class Person(val name: String, val age: Int) {
    fun isAdult(): Boolean = age >= 19
}

fun main() {
    val alice = Person("Alice", 25)

    // alice 객체에 '묶인' isAdult 멤버 함수 참조
    val isAliceAdult: () -> Boolean = alice::isAdult

    println(isAliceAdult()) // 수신 객체(alice) 없이 호출 가능 -> 출력: true
}
```
