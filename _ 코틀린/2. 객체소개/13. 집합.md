# 코틀린의 집합 (Set)
#Kotlin/Collection

> [!INFO] 집합(Set)이란?
> **집합(Set)**은 코틀린의 기본 컬렉션 타입 중 하나로, **순서가 없으며 중복된 원소를 허용하지 않는** 데이터의 모음입니다. 수학의 '집합'과 동일한 특성을 가집니다.

`Set`은 특정 원소의 존재 여부를 확인하거나, 중복을 제거하는 용도로 매우 유용합니다.

## 1. 집합의 주요 특징

1.  **중복 불허**: 동일한 원소는 단 하나만 저장될 수 있습니다.
    ```kotlin
    val numbers = setOf(1, 2, 2, 3, 3, 3)
    println(numbers) // 출력: [1, 2, 3] (중복이 자동으로 제거됨)
    ```
2.  **순서 없음**: `List`와 달리, 원소들이 저장된 순서를 보장하지 않습니다. 따라서 인덱스로 원소에 접근할 수 없습니다.

## 2. 집합 생성

### 불변 집합 (`Set`)
`setOf()` 함수를 사용하여 생성하며, 한 번 생성하면 내용을 변경할 수 없습니다.
```kotlin
val fruits: Set<String> = setOf("Apple", "Banana", "Cherry")
```

### 가변 집합 (`MutableSet`)
`mutableSetOf()` 함수를 사용하여 생성하며, 원소를 추가하거나 삭제할 수 있습니다.
```kotlin
val numbers: MutableSet<Int> = mutableSetOf(1, 2, 3)
numbers.add(4)      // 원소 추가
numbers.remove(2)   // 원소 삭제
println(numbers)    // 출력: [1, 3, 4]
```

## 3. 집합의 기본 사용법

-   **포함 여부 확인**: `in` 키워드를 사용합니다. `Set`에서 가장 빠르고 효율적인 연산입니다.
    ```kotlin
    val fruits = setOf("Apple", "Banana", "Cherry")
    println("Apple" in fruits) // 출력: true
    ```
-   **크기 확인**: `size` 프로퍼티를 사용합니다.
    ```kotlin
    println(fruits.size) // 출력: 3
    ```
-   **반복**: `for` 루프나 `forEach` 함수를 사용합니다.
    ```kotlin
    for (fruit in fruits) {
        println(fruit)
    }
    ```

## 4. 집합 연산

수학의 집합처럼 합집합, 교집합, 차집합 연산을 직관적으로 수행할 수 있습니다.

```kotlin
val setA = setOf(1, 2, 3, 4)
val setB = setOf(3, 4, 5, 6)

// 합집합
val union = setA.union(setB) // 또는 setA union setB
println(union) // 출력: [1, 2, 3, 4, 5, 6]

// 교집합
val intersect = setA.intersect(setB) // 또는 setA intersect setB
println(intersect) // 출력: [3, 4]

// 차집합
val subtract = setA.subtract(setB) // 또는 setA subtract setB
println(subtract) // 출력: [1, 2]
```

## 5. 리스트와 집합의 변환

`toList()`, `toSet()` 등의 함수를 사용하여 컬렉션 타입을 쉽게 변환할 수 있습니다.

```kotlin
val numberList = listOf(1, 2, 2, 3, 3, 3)

// 리스트에서 중복을 제거하는 가장 쉬운 방법
val numberSet = numberList.toSet()
println(numberSet) // 출력: [1, 2, 3]
```
